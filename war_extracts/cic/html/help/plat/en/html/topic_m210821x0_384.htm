Display virtual switches of a resource pool

Perform this task to display virtual switches in a resource pool.

Procedure

  1. From the left navigation pane, select VDC > Resource Pools > Resource pool name.

  1. Click the vSwitches tab.

Parameters

  • Forwarding Mode: Forwarding mode of the virtual switch.

  • VEB: In this mode, the virtual switch forwards traffic between VMs based on software.

  • VXLAN (SDN): Forwarding mode provided by the VXLAN solution based on SDN controllers and cloud computing management platforms.

  • Action: Actions that operators can perform on the virtual switch. To delete the virtual switch, click the  icon.

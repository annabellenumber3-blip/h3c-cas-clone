Add virtual switches to a resource pool

Perform this task to add virtual switches to a resource pool.

Restrictions and guidelines

You can add a virtual switch to a resource pool only when all hosts in the corresponding cluster have the same name and forwarding settings for the virtual switch.

Procedure

  1. From the left navigation pane, select VDC > Resource Pools > Resource pool name.

  1. Click the vSwitches tab.

  1. Click Add vSwitch.

  1. In the dialog box that opens, select one or multiple virtual switches.

  1. Click OK.

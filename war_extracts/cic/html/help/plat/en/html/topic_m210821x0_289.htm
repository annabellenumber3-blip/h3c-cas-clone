Display detailed information about an operator

Perform this task to display detailed information about an operator.

Procedure

  1. From the left navigation pane, select System > Operators & Groups > Operators.

  1. Select the target operator, and then click View Operator Info.

Parameters

  • Login Name: Login name used by the operator to log in to CIC.

  • Authentication: Authentication method.

  • Password Authentication: Local authentication is performed using an operator username and password.

  • LDAP Authentication: LDAP authentication is performed using an operator username and password.

  • LDAP Server: Name of the LDAP server. This field is displayed if you select LDAP Authentication.

  • Operator: Name of the operator.

  • Operator Group: Operator group to which the operator belongs.

  • Email: Email address of the operator.

  • Phone: Phone number of the operator.

  • Access Control Policy: Access control policy.

  • Online Operators: Maximum number of users that can use this operator account to log in to the system.

  • Permission Settings: Select the actions that the operator can perform. These actions are exclusive to this operator. An operator has all privileges of the operator group after you specify an operator group for the operator.

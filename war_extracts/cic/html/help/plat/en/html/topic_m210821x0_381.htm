Add storage resources in a resource pool

Perform this task to add storage pools in a resource pool.

Procedure

  1. From the left navigation pane, select VDC > Resource Pools > Resource pool name.

  1. Click the Storage Resources tab.

  1. Click the  icon. In the dialog box that opens, select storage pools and then click OK.

Parameters

  • Storage Pool Alias: Storage pool alias.

  • Type: Storage pool type.

  • Available: Available storage size.

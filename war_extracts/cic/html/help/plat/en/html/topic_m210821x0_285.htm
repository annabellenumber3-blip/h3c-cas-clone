Add an operator

Perform this task to add an operator.

Restrictions and guidelines

The actual privileges of an operator are a combination of the privileges of the operator and the operator group to which the operator belongs.

Procedure

  1. From the left navigation pane, select System > Operators & Groups > Operators.

  1. Click Add Operator.

  1. Enter the login name, operator name, email address, phone number, and number of online operators, and select an operator group and access control policy.

  1. Select an authentication method. Options include Password Authentication and LDAP Authentication.

  • If you select Password Authentication, enter the login password and confirm password, and then click Next.

  • If you select LDAP Authentication, select an LDAP server from the LDAP Server list, and then click Next.

  1. Enable or disable operator as needed.

  1. Click Finish.

Parameters

  • Login Name: Enter the login name used by the operator to log in to CIC.

  • Authentication: Select an authentication method. Options include Password Authentication and LDAP Authentication.

  • Password Authentication: Local authentication is performed using an operator username and password.

  • LDAP Authentication: LDAP authentication is performed using an operator username and password.

  • Password: Enter the password that the user uses to log in to CIC. The password must conform to the password policy. This field is required if you select Password Authentication.

  • Confirm: Enter the confirm password. It must be the same as the login password. This field is required if you select Password Authentication.

  • LDAP Server: Select an LDAP server. This field is required if you select LDAP Authentication.

  • Operator Group: Select the operator group to which the operator belongs. The default operator groups are system administrator group and organization administrator group.

  • Email: Enter the email address of the operator. The email address must be a valid address, account@domain.com, for example.

  • Phone: Enter the phone number of the operator.

  • Access Control Policy: Select an access control policy.

  • Online Operators: Select the maximum number of users that can log in to the system by using this operator account. 0 means no limit.

  • Disable Operator: If you disable an operator, the operator cannot log in to the system.

  • Permission Settings: Select the actions that the operator can perform. These actions are exclusive to this operator. An operator has all privileges of the operator group after you specify an operator group for the operator.

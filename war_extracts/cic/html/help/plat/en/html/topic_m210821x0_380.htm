Display storage resources of a resource pool

Perform this task to display storage pool information for a resource pool.

Procedure

  1. From the left navigation pane, select VDC > Resource Pools > Resource pool name.

  1. Click the Storage Resources tab.

  1. Select a resource pool.

Parameters

  • Basic Attributes

  • Alias: Storage pool alias.

  • Type: Storage pool type.

  • Total: Total size of the shared storage pool.

  • Available: Available size of the shared storage pool.

  • Used: Used size of the shared storage pool.

  • Storage Usage: Proportion of used storage size to total storage size.

  • Top 5 Organizations by Storage Allocation: Storage size bar graph for the top 5 organizations with the largest storage size for the storage pool.

  • Top 5 VMs by Storage Size: Storage size bar graph for the top 5 VMs with the largest storage size for the storage pool.

  • I/O Throughput: I/O throughput trend graph for each disk. The horizontal axis represents the time, and the vertical axis represents the I/O throughput in KBps.

  • IOPS: Disk request trend graph for each disk. The horizontal axis represents the time, and the vertical axis represents the number of read and write operations per second.

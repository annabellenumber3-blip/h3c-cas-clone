Import users

Perform this task to import users in bulk so that operators can add users to CIC in bulk.

Restrictions and guidelines

  • To successfully add a user, make sure the column names corresponding to the login name, login password, username, and email are the same as those in the user information file and the information in the user information file is correct.

  • To successfully add a user, make sure the separator you select is the same as the delimiter in the user information file.

  • The size of the user information file cannot exceed 30 M. The maximum number of rows and columns are 5000 and 30, respectively, and the number of columns for each row must be the same.

  • A user cannot be added if the login name and email in the user information file to upload are the same as those in the user list in CIC.

Procedure

  1. From the left navigation pane, select VDC > Users & Groups > Users.

  1. Click Import.

  1. In the dialog box that opens, click Browse to select the target file, and then click Open.

  1. To preview the user information, click Preview.

  1. Select a delimiter, select the start row, and then click Next.

  1. Click the  icon to the right of the Organization field to select an organization.

  1. Select a login name, login password, username, identity number, email, phone number, contact address, and user group.

  1. Click Finish.

Parameters

  • Import File: Click to import a file. Operators can create a file, or click Template File to download a template and configure user information as required. Txt and CSV files separated by commas, semicolons, the Tab key, and spaces are supported.

  • Delimiter: Select a separator. Options include Comma (,), Semicolon (;), Tab, and Space.

  • Start Row: Specify a start row for importing user data.

  • Organization: Select an organization for the user.

  • Login Name: Select a column corresponding to the login name.

  • Login Password: Select a column corresponding to the login password.

  • Username: Select a column corresponding to the username.

  • Identity No: Select a column corresponding to the identity number.

  • Email: Select a column corresponding to the email address.

  • Phone Number: Select a column corresponding to the phone number.

  • Contact: Select a column corresponding to the contact address.

  • User Group: Select a column corresponding to the user group name.

Filter users

Operators can filter users by setting filtering criteria.

Procedure

  1. From the left navigation pane, select VDC > Users & Groups > Users.

  1. Click Advanced Filter.

  1. In the dialog box that opens, set the filtering criteria as required.

  1. Click OK.

All users matching the criteria will be displayed on the list.

Parameters

  • Login Name: Login name that the user uses to log in to CAS SSV.

  • Authentication Type: Type of authentication that the user uses to log in to CAS SSV.

  • Username: Username of the user.

  • Identity No.: Identify number of the user.

  • User Group: User group to which the user belongs, including all user groups that the operator can manage.

  • LDAP Policy: Name of the LDAP synchronization policy.

  • Email: Email address of the user.

  • Organization: Organization to which the user belongs.

  • Phone Number: Phone number of the user.

  • User State: State of the user, including Valid and Invalid.

  • Contact: Contact address of the user.

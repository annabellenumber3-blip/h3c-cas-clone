Display users

Perform this task to display information about all users in CIC.

Procedure

From the left navigation pane, select VDC > Users & Groups > Users.

Parameters

  • Login Name: Login name that the user uses to log in to CAS SSV.

  • Username: Username of the user.

  • User Group: User group to which the user belongs.

  • Authentication Type: Type of authentication that the user uses to log in to CAS SSV.

  • LDAP Policy: Name of the LDAP synchronization policy.

  • Last Login: Time at which the user logged in to CAS SSV for the last time.

  • Email: Email address of the user.

  • Organization: Organization to which the user belongs.

  • Phone Number: Phone number of the user.

  • User State: State of the user.

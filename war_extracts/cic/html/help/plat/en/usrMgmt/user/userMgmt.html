User management

A user in CIC is a terminal user created by an operator in CIC. A terminal user can log in to CAS SSV to request and manage VM resources. Operators can add, modify, or delete a user or display detailed information about a user.

A terminal user can use the username and password created by an operator to access the CAS SSV at http://<CIC IP address>:8080/ssv/ or https://<CIC IP address>:8443/ssv/.

Features

  • Display users

  • Add a user

  • Modify a user

  • Delete users

  • Filter users

  • Display user details

  • Import users

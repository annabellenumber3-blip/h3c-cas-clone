Add a user

Perform this task to add a user.

Procedure

  1. From the left navigation pane, select VDC > Users & Groups > Users.

  1. Click Add.

  1. Enter the login name, login password, confirm password, username, identity number, email, phone number, and contact, and select the organization and user group to which the user belongs.

  1. Click OK.

Parameters

  • Login Name: Enter the login name that the user uses to log in to CAS SSV.

  • Authentication Type: Type of authentication that the user uses to log in to CAS SSV.

  • Login Password: Enter the password that the user uses to log in to CAS SSV.

  • Confirm: Enter the password again. It must be the same as the login password.

  • Username: Enter the username of the user.

  • Identity No.: Enter the identify number of the user, such as ID card number and passport number.

  • Email: Enter the email address of the user. The email address must be a valid address, account@domain.com, for example.

  • Organization: Select the organization to which the user belongs.

  • Phone Number: Enter the phone number of the user.

  • Contact: Enter the contact address of the user.

  • User Group: Select the user group to which the user belongs.

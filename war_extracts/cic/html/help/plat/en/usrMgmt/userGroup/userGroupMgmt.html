User group management

A user group is a collection of users. It can contain multiple levels of subgroups. Users in a user group or subgroup have the same access right. With user group management, operators can add, modify, or delete a user group/subgroup or display detailed information about a user group.

Users in the same user group or subgroup can use the VM resources of the group after logging in to CIC.

Features

  • Display user groups

  • Add a user group

  • Add a subgroup

  • Modify a user group or subgroup

  • Delete a user group or subgroup

  • Display user group details

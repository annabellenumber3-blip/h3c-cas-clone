Display user group details

Perform this task to view detailed information about a user group.

Procedure

  1. From the left navigation pane, select VDC > Users & Groups > User Groups.

  1. For a parent group, click the  icon in the Action column. For a subgroup, click the  icon to the left of its parent group name, and then click the  icon in the Action column.

Parameters

  • Parent Group: Parent group of the subgroup. If this field displays /, the current user group is a parent group.

  • Name: Name of the user group.

  • Organization: Organization to which the user group belongs.

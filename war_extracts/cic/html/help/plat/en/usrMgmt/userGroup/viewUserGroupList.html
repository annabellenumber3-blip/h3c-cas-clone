Display user groups

Perform this task to display information about all user groups that the current operator can manage, including parent groups and their subgroups. Operators can display subgroups of a parent group by clicking  before the parent group name.

Procedure

From the left navigation pane, select VDC > Users & Groups > User Groups.

Parameters

  • Name: Name of the user group.

  • Organization: Organization to which the user group belongs.

  • Action: Actions that operators can perform on the user group.

  • To add a subgroup, click the  icon.

  • To modify a user group, click  icon.

  • To delete a user group, click the  icon.

  • To display user group information, click the  icon.

Modify a user group or subgroup

Perform this task to modify a user group or subgroup.

Procedure

  1. From the left navigation pane, select VDC > Users & Groups > User Groups.

  1. For a parent group, click the  icon in the Action column. For a subgroup, click the   icon to the left of its parent group name, and then click the  icon in the Action column.

  1. Enter the user group name and description, and select an organization.

  1. Click OK.

Parameters

  • Parent Group: Parent group of the subgroup. This field displays / if no parent group exists.

  • Name: Enter the name of the user group. A user group name must be unique in an organization. User groups in different organizations can have the same name.

  • Organization: Select the organization to which the user group belongs.

  • For a parent group that does not contain any subgroups, this parameter can be modified.

  • For a user group that contains users, a parent group that contains subgroups, or a subgroup, this parameter cannot be modified.

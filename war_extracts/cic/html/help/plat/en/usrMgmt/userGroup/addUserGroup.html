Add a user group

Perform this task to add a user group.

Procedure

  1. From the left navigation pane, select VDC > Users & Groups > User Groups.

  1. Click Add Group.

  1. Enter the user group name and description, and select an organization.

  1. Click OK.

Parameters

  • Parent Group: If this field displays /, the current user group is a parent group.

  • Name: Name of the user group. A user group name must be unique in an organization. User groups in different organizations can have the same name.

  • Organization: Select the organization to which the user group belongs.

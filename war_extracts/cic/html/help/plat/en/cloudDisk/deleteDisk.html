Delete cloud disks

Perform this task to delete cloud disks.

Procedure

  1. From the left navigation pane, select Cloud Services > Cloud Disks.

  1. To delete cloud disks one by one, click the  icon in the Action column for the target cloud disk, and then click OK in the dialog box that opens.

  1. To delete multiple cloud disks in bulk, select target cloud disks and click Batch Delete and then click OK in the dialog box that opens.

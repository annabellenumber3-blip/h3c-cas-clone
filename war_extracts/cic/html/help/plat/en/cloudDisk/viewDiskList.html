Display cloud disks

Perform this task to display all cloud disks.

Procedure

From the left navigation pane, select Cloud Services > Cloud Disks.

Parameters

  • Name: Cloud disk name.

  • State: Cloud disk state.

  • Path: Mount path of the corresponding storage on a host.

  • Size: Cloud disk size.

  • Organization: Organization of the user who uses the cloud disk.

  • Cloud Host: Cloud host name.

  • User: User who uses the cloud disk.

  • Action: Actions that operators can perform on the cloud disk. To delete the cloud disk, click the  icon.

Bulk deploy VMs for an organization in a vCenter cloud resource

Perform this task to bulk deploy VMs by using a VM template for an organization in a vCenter cloud resource. You can specify the VM assignment mode when deploying the VMs.

Restrictions and guidelines

If the port profile of a VM template is changed, VMs deployed by using the template use the new profile.

Procedure

  1. From the left navigation pane, select VDC > Organizations > Organization name.

  1. Click the Templates tab.

  1. Click the  icon in the Action column for a VM template.

  1. Select a resource pool.

  1. Select a deployment method.

  • Manually Defined: Manually configure VMs one by one. For details, see Deploy a VM for an organization in a vCenter cloud resource.

  • Import from File: Import VM information from a file. For details, see Import VM configurations from a file.

  1. Click OK.

Parameters

Resource Pool Name: Select a resource pool.

Manage VM templates of an organization

Perform this task to manage the VM templates of an organization. You can view network information and storage information about a VM template, add a VM template, deploy VMs one by one, bulk deploy VMs, and delete VM templates.

Features

  • Display VM templates of an organization

  • Display network and storage information for a VM template

  • Synchronize VM templates of an organization

  • Deploy a VM for an organization in a vCenter cloud resource

  • Bulk deploy VMs for an organization in a vCenter cloud resource

  • Import VM configurations from a file

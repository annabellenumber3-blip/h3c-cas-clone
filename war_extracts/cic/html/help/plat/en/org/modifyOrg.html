Modify an organization

Perform this task to modify an organization.

Procedure

  1. From the left navigation pane, select VDC > Organizations.

  1. Click the  icon in the Action column for an organization.

  1. In the dialog box that opens, enter the description and alias prefix, specify the resource quotas, including the number of VMs, the number of CPU cores, the memory size, and the storage size, and set whether to allow modifications to VMs.

  1. Click OK.

Parameters

  • Alias Prefix: Enter the alias prefix of the VMs that will be created in the organization.

  • VMs: Enter the maximum number of VMs in the organization. The value of this parameter cannot be smaller than the number of existing VMs in the organization.

  • CPUs: Enter the maximum number of CPU cores for the organization. The value of this parameter cannot be smaller than the number of assigned CPU cores of the organization.

  • Memory: Specify the maximum memory size for the organization. The value of this parameter cannot be smaller than the assigned memory size of the organization.

  • Storage: Specify the maximum storage size for the organization. The value of this parameter cannot be smaller than the assigned storage size of the organization.

  • Modification: Set whether to allow modifications to VMs during VM deployment.

Manage users of an organization

In CIC, a user is a terminal user of the VMs in an organization. After an organization is created, an organization administrator must create users and user groups for the organization. The system provides a user list for organization administrators to manage all users of an organization.

Features

  • Display users of an organization

  • Display VMs assigned to a user in an organization

  • Display cloud backup polices of a user in an organization

  • Display details about a user in an organization

  • Add a user for an organization

  • Modify a user for an organization

  • Delete a user from an organization

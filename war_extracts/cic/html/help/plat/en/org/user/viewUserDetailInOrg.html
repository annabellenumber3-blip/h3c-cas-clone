Display details about a user in an organization

Perform this task to display details about a user in an organization.

Procedure

  1. From the left navigation pane, select VDC > Organizations > Organization name.

  1. Click the Users tab.

  1. Click the  icon for a user.

Parameters

  • Login Name: Login name that the user uses to log in to SSV.

  • Authentication Type: SSV login authentication mode. Only password authentication mode (based on login name and password) is used.

  • Username: Username of the user.

  • Identity No.: Identify number of the user, such as the ID number or passport number.

  • Email: Email address of the user.

  • Organization: Organization of the user.

  • Phone Number: Phone number of the user.

  • Contact: Contact address of the user.

  • User Group: User group to which the user belongs.

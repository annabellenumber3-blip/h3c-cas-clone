Modify a user for an organization

Perform this task to modify a user for an organization.

Restrictions and guidelines

The login names of users must be unique in CIC.

Procedure

  1. From the left navigation pane, select VDC > Organizations > Organization name.

  1. Click the Users tab.

  1. Click the  icon for a user.

  1. In the dialog box that opens, enter the login name, password, username, identity number, email address, phone number, and contact address, and select an organization and a user group.

  1. Click OK.

Parameters

  • Login Name: Enter the login name that the user uses to log in to SSV.

  • Authentication: SSV login authentication mode. Only password authentication mode (based on login name and password) is used.

  • Login Password: Enter the password that the user uses to log in to SSV.

  • Confirm: Confirm the password.

  • Username: Enter the username of the user.

  • Identity No.: Enter the identify number of the user, such as the ID number or passport number.

  • Email: Enter the email address of the user, in the account@domain.com format.

  • Organization: Select an organization.

  • Phone Number: Enter the phone number of the user.

  • Contact: Enter the contact address of the user.

  • User Group: Select the user group to which the user belongs.

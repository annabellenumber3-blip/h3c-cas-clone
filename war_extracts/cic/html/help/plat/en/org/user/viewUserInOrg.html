Display users of an organization

Perform this task to display all users of an organization.

Procedure

  1. From the left navigation pane, select VDC > Organizations > Organization name.

  1. Click the Users tab.

Parameters

  • Login Name: Login name that the user uses to log in to SSV.

  • Username: Username of the user.

  • User Group: User group to which the user belongs.

  • Authentication: SSV login authentication mode. Only password authentication mode (based on login name and password) is used.

  • Email: Email address of the user.

  • Phone Number: Phone number of the user.

  • Action: Actions that operators can perform on the user.

  • To modify the user, click the  icon.

  • To view details about a user, click the  icon.

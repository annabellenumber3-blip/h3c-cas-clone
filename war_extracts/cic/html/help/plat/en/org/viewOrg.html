Display organization summary

Perform this task to view summary information about an organization, including VM state statistics, user statistics, CPU/memory statistics, storage/VM statistics, top 5 VMs by CPU usage, and top 5 VMs by memory usage.

Procedure

From the left navigation pane, select VDC > Organizations > Organization name.

Parameters

  • VM State Statistics: Total number of VMs in the organization, and the number of VMs in each state.

  • User Statistics: Total number of users in the organization, and the number of users in each state.

  • CPU/Memory Statistics: CPU and memory statistics.

  • Quota: Maximum number of CPUs that can be used by the organization.

  • Assigned: Current number of CPUs assigned to the VMs of the organization.

  • Overcommit: Number of assigned CPUs divided by CPU quota.

  • Quota: Maximum memory size that can be used by the organization.

  • Assigned: Current memory size assigned to the VMs of the organization.

  • Overcommit: Assigned memory size divided by memory quota.

  • Storage/VM Statistics: Storage and VM statistics.

  • Quota: Maximum storage size that can be used by the organization.

  • Assigned: Current storage size assigned to the VMs of the organization.

  • Overcommit: Assigned storage size divided by storage quota.

  • Quota: Maximum number of VMs that the organization can manage.

  • Assigned: Current number of VMs assigned to the organization.

  • Ratio: Number of assigned VMs divided by VM quota.

  • Top 5 VMs by CPU Usage: CPU usage ring graph for the top 5 VMs with the highest CPU usage.

  • Top 5 VMs by Memory Usage: Memory usage bar graph for the top 5 VMs with the highest memory usage.

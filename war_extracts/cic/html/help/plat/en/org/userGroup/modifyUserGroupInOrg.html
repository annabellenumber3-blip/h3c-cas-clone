Modify a user group for an organization

Perform this task to modify a user group or subgroup.

Procedure

  1. From the left navigation pane, select VDC > Organizations > Organization name.

  1. Click the User Groups tab.

  1. If a user group is not a subgroup, click the  icon in the Action column for the user group. To modify a subgroup, click the  icon for the parent group, and then click the  icon in the Action column for the subgroup.

  1. In the dialog box that opens, enter the user group name and description.

  1. Click OK.

Parameters

  • Parent Group: Parent group of the user group. If the user group does not have a parent group, a forward slash / is displayed.

  • Organization: Organization to which the user group belongs.

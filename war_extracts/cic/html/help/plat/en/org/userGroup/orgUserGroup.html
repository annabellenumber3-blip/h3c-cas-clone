Manage user groups of an organization

In CIC, a user group is a group of terminal users of the VMs in an organization. After an organization is created, an organization administrator must create users and user groups for the organization. If a VM is assigned to a user group, all users in the user group can use the VM. The system provides a user group list for organization administrators to manage all user groups of an organization.

Features

  • Display user groups of an organization

  • Display VMs assigned to a user group in an organization

  • Display details about a user group in an organization

  • Add a user group for an organization

  • Modify a user group for an organization

  • Add a subgroup in a user group for an organization

  • Delete a user group from an organization

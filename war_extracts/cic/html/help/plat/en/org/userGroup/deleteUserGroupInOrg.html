Delete a user group from an organization

Perform this task to delete a user group or subgroup from an organization.

Restrictions and guidelines

You cannot delete a user group if it contains subgroups.

Procedure

  1. From the left navigation pane, select VDC > Organizations > Organization name.

  1. Click the User Groups tab.

  1. If a user group is not a subgroup, click the  icon in the Action column for the user group. For a subgroup, click the  icon for the parent group, and then click the  icon in the Action column for the subgroup.

  1. In the dialog box that opens, click OK.

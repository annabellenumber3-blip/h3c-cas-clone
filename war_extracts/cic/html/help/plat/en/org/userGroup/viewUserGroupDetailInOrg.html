Display details about a user group in an organization

Perform this task to display details about a user group in an organization.

Procedure

  1. From the left navigation pane, select VDC > Organizations > Organization name.

  1. Click the User Groups tab.

  1. If a user group is not a subgroup, click the  icon in the Action column for the user group. To display details about a subgroup, click the  icon for the parent group, and then click the  icon in the Action column for the subgroup.

Parameters

  • Parent Group: Parent group of the user group. If the user group does not have a parent group, a forward slash / is displayed.

  • Name: Name of the user group.

  • Organization: Organization to which the user group belongs.

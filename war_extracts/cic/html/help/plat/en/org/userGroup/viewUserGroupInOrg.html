Display user groups of an organization

Perform this task to display all user groups of an organization.

Procedure

  1. From the left navigation pane, select VDC > Organizations > Organization name.

  1. Click the User Groups tab.

Parameters

  • Name: Name of the user group.

  • Action: Actions that operators can perform on the user group.

  • To add a subgroup for the user group, click the  icon.

  • To modify the user group, click the  icon.

  • To delete the user group, click the  icon.

  • To display details about the user group, click the  icon.

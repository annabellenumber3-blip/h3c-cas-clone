Add an organization

Perform this task to add an organization.

Restrictions and guidelines

An organization administrator is an operator that manages the resources of an organization. Organization administrators are created by a system administrator. For information about the creation process, see Add an operator.

Procedure

  1. From the left navigation pane, select VDC > Organizations.

  1. Click Add.

  1. In the dialog box that opens, enter the organization name, description, and alias prefix. Set whether to allow modifications to VMs as needed.

  1. Click the  icon, select one or multiple resource pools, and then click Next.

  1. Click Select Administrator or Select Administrator Group, select administrators or administrator groups, and then click Next.

  1. Specify the resource quotas, including the number of VMs, the number of CPU cores, the memory size, and the storage size.

  1. Click Finish.

Parameters

Basic

  • Alias Prefix: Enter the alias prefix of the VMs that will be created in the organization.

  • Modification: Set whether to allow modifications to VMs during VM deployment.

  • Resource Pool: Select resource pools to provide compute, network, and storage resources for VMs in the organization.

  • VMs: Enter the maximum number of VMs in the organization.

  • CPUs: Enter the maximum number of CPU cores for the organization.

  • Memory: Specify the maximum memory size for the organization.

  • Storage: Specify the maximum storage size for the organization.

Manage VMs of an organization

Perform this task to manage the VMs of an organization. As an organization administrator, you can perform the following operations on VMs: start, suspend, resume, hibernate, reboot, shut down, power off, assign, manage snapshots, back up, delete, operate the console, and view user or user group information.

Features

  • Display VMs of an organization

  • Display user or user group information for a VM

  • Assign a VM to users or user groups

  • Revoke a VM

  • Start a VM

  • Suspend a VM

  • Resume a VM

  • Hibernate a VM

  • Restart a VM

  • Shut down a VM

  • Power off a VM

  • Delete a VM

  • Back up a VM immediately

  • Manage VM snapshots

  • Open the Web console of a VM

  • Manage a VM through the Web console

  • Install an OS on a VM

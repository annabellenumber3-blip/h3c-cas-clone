Display user or user group information for a VM

Perform this task to display the user or user group information for a VM.

Procedure

  1. From the left navigation pane, select VDC > Organizations > Organization name.

  1. Click the VMs tab.

  1. Select a VM. The Users/User Groups area displays the users or user groups that can use the VM.

Parameters

  • Login Name/Group Name: Login name that a user uses to log in to SSV, or the name of a user group. This parameter is empty if the VM is not assigned to any user or user group.

  • Username/Description: Username of a user, or the description of a user group. This parameter is empty if the VM is not assigned to any user or user group.

  • Type: Whether the item indicates an organization, user, or user group.

  • Virtual Desktop Pool Name: Virtual desktop pool to which the VM belongs.

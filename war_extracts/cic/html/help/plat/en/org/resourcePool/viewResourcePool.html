Display resource pools of an organization

Perform this task to display resource pools of an organization.

Procedure

  1. From the left navigation pane, select VDCs > Organizations > Organization name.

  1. Click the Resource Pools tab.

Parameters

  • Basic Attributes

  • Cloud Resource: Alias of the cloud resource that provides the resource pool.

  • Description: Description of the resource pool.

  • CPUs: Number of physical CPU cores in the resource pool.

  • Memory: Memory size of the resource pool.

  • Storage: Storage size of the resource pool.

  • Available Storage: Available storage size of the resource pool.

  • Compute Resource Monitoring Info: CPU usage and memory usage of the resource pool.

  • CPU: Host CPU usage dashboard graph.

  • Memory: Host memory usage dashboard graph.

  • Storage Resource Monitoring Info: Storage usage pie graph of the resource pool.

  • Top 5 VMs by Allocated CPUs (cores): Bar graph for the top 5 VMs with the most CPU cores.

  • Top 5 VMs by Allocated Memory Size (MB): Bar graph for the top 5 VMs with the most memory.

  • Top 5 VMs by Allocated Storage Size (GB): Bar graph for the top 5 VMs with the most storage.

Publish a resource pool for an organization

Perform this task to publish a resource pool for an organization.

Procedure

  1. From the left navigation pane, select VDC > Organizations > Organization name.

  1. Click the Resource Pools tab.

  1. Click the  icon, or click Publish Resource Pools.

  1. In the dialog box that opens, select a resource pool and click OK.

Parameters

  • Cloud Resource: Alias of the cloud resource that provides the resource pool.

  • Cloud Resource Type: Cloud resource type.

  • Compute Resource Monitoring Info: CPU usage and memory usage of the resource pool.

  • Basic Attributes

  • CPUs: Number of physical CPU cores in the resource pool.

  • Memory: Memory size of the resource pool.

  • Storage: Storage size of the resource pool.

  • Available Storage: Available storage size of the resource pool.

  • Storage Usage: Storage usage of the resource pool.

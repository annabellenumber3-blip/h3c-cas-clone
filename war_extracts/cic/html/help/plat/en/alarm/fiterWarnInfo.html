Filter realtime alarm information

Perform this task to filter realtime alarm information.

Procedure

  1. From the left navigation pane, select Alarms.

  1. Click Filter.

  1. Select a confirmation state, alarm level, alarm type, and time range, enter the alarm content and source, and then click OK.

  1. To reset the filtering criteria, click Reset.

Parameters

  • Confirmation State: Select the confirmation state of the alarm, which indicates whether the alarm has been processed. Options include All, Confirmed, and Unconfirmed.

  • Alarm Level: Select the level of the alarm. Options include All, Critical, Major, Minor, and Tip.

  • Alarm Type: Select the type of the alarm. Options include All, Host Resource Alarm, VM Resource Alarm, Cluster Resource Alarm, Fault Alarm, and Other Alarm.

  • Alarm Content: Enter the content of the alarm.

  • Alarm Source: Enter the source of the alarm.

  • From: Select the start time for filtering alarms.

  • To: Select the end time for filtering alarms.

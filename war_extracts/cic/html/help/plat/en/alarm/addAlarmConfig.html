Add alarm configuration

Perform this task to add alarm configuration.

Procedure

  1. From the left navigation pane, select Alarms > Configurations.

  1. Click Add Alarm Configuration.

  1. In the dialog box that opens, select a cloud resource, mail notification template, and SMS notification template, enable binding the same alarm item as needed , and then click Next.

  1. Select the concerned items, such as hosts and VMs.

  1. Click Finish.

Parameters

  • Cloud Resource: Select a source for the alarm information.

  • Mail Notification Template: Select a mail notification template.

  • SMS Notification Template: Select an SMS notification template.

  • Same Alarm Item Binding: Enable or disable binding the same alarm items.

  • Alarm Items: Select one or multiple sources of the alarms, hosts or VMs. If you do not select any items, the source is not limited.

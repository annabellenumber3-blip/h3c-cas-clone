Add an alarm notification template

Perform this task to add an alarm notification template.

Procedure

  1. From the left navigation pane, select Alarms > Notification Templates.

  1. Click Add Notification Template.

  1. In the dialog box that opens, enter the template name and description, select the template type. Options include Mail Notification and SMS Notification.

  1. Click Next.

  1. Enter the destination address, click the  icon to add the address to the destination address list, and then click Next. To delete a destination address, select the target address in the list, and then click the  icon.

  1. Select one or multiple alarm levels.

  1. Select alarm rules. If you select All, all system-defined alarm rule will be used. If you select Selected Alarms, select target alarm rules in the alarm rule list.

  1. Click Finish.

Parameters

  • Type: Select a notification type. Options include Mail Notification and SMS Notification.

  • Destination: Enter the destination of the notification. If you select Mail Notification, enter the destination mail address. If you select SMS Notification, enter the destination phone number.

  • Alarm Level: Select alarm levels. Options include Critical, Major, Minor, and Tip.

  • Alarm Rule: Select alarm rules. Options include All and Selected Alarms.

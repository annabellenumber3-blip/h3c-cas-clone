Display alarm configuration

Perform this task to display alarm configurations available in CIC.

Procedure

From the left navigation pane, select Alarms > Configurations.

Parameters

  • Cloud Resource: Name of the cloud resource.

  • Mail Notification Template: Name of the mail notification template configured for the cloud resource.

  • SMS Notification Template: Name of the SMS notification template configured for the cloud resource.

  • Action: Actions that operators can perform on the alarm configuration.

  • To modify the alarm configuration, click the  icon.

  • To deploy the alarm configuration, click the  icon.

  • To delete the alarm configuration, click the  icon.

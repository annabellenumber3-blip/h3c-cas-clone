Display realtime alarm information

Perform this task to display realtime alarm information for all cloud resources in CIC. Operators can confirm alarms and filter concerned alarms.

Procedure

  1. From the left navigation pane, select Alarms > cloud resource name.

  1. Select a refresh interval as needed. The system will refresh the alarm information at the selected interval.

Parameters

  • Level: Level of the alarm.

  • Confirmation State: Confirmation state of the alarm.

  • Alarm Source: Source of the alarm, which displays the device type, name, and IP address.

  • Type: Type of the alarm.

  • Alarm Content: Content of the alarm.

  • Generated At: Time when the alarm was generated.

  • Refresh: Interval at which the alarm information is refreshed.

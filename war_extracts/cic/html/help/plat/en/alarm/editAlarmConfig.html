Modify alarm configuration

Perform this task to modify alarm configuration.

Procedure

  1. From the left navigation pane, select Alarms > Configurations.

  1. Click the  icon in the Action column for the target alarm configuration.

  1. In the dialog box that opens, select a cloud resource, mail notification template, SMS notification template, and enable or disable binding the same alarms as needed, and then click Next.

  1. Select alarm items, hosts or VMs as needed.

  1. Click Finish.

Parameters

  • Cloud Resource: Select a source for the alarm information.

  • Mail Notification Template: Select a mail notification template.

  • SMS Notification Template: Select an SMS notification template.

  • Same Alarm Item Binding: Enable or disable binding the same alarm items as needed.

  • Alarm Items: Select one or multiple sources of the alarms, hosts or VMs. If you do not select any items, the source is not limited.

Manage cloud backup

Perform this task to display backup information for cloud hosts.

Procedure

From the left navigation pane, select Cloud Services > Cloud Backup.

Parameters

  • Backed Up Cloud Host: Cloud host the data of which is backed up.

  • Backup Time: Time when the backup was performed.

  • Backup Path: Path for saving the backup.

  • User: User who uses the cloud host. After the user requests a backup policy and the administrator of the organization issues the backup policy, the system backs up data of the cloud host based on the backup policy.

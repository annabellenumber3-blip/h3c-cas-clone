Quick Start

This quick start describes the basic tasks that you can perform with CIC.

  1. Managing operators

As a best practice to ensure security, change the default password for the admin operator at the first login, as described in Modify an operator. Then, create operators with different privileges and permissions, as described in Add an operator.

  1. Adding vCenter cloud resources

CIC has a Cloud Resources menu in the navigation tree, which is empty by default. You must add vCenter cloud resources to this menu before you can allocate them. To add cloud resources, see Add a vCenter cloud resource.

  1. Managing vCenter cloud resources

To manage vCenter resources, see Manage vCenter cloud resources.

  1. Adding organizations

CIC abstracts the virtualized resource pools provided by CVM into computing, storage, and network services provisioned on a per-organization basis. Organizations are definable depending on the business needs, which can be departments in the private cloud of an enterprise or be external customers on a public cloud. To add organizations, see Add an organization.

Add a resource pool

Perform this task to add a resource pool.

Restrictions and guidelines

  • Compute resources are assigned on a per-cluster basis. A cluster provides HA features for VMs in an organization.

  • A virtual switch can be selected only when it is used by all hosts in the selected cluster.

  • The storage pools you select must be shared file systems of the selected cluster.

  • The backup storage pool is used for storing cloud host backup files of users in an organization. You can configure only one backup storage pool.

Procedure

  1. From the left navigation pane, select VDC > Resource Pools.

  1. Click Add Resource Pool.

  1. In the dialog box that opens, enter the resource pool name, description, select a cloud resource, select whether to use local storage, and then click Next.

  1. Select a cluster and then click Next.

  1. Select one or multiple virtual switches and then click Next.

  1. Select one or multiple storage pools and specify the backup storage pool for the resource pool as needed.

  1. Click Finish.

Parameters

  • Resource Pool Name: Enter the name of the resource pool.

  • Description: Enter the description of the resource pool.

  • Cloud Resource: Select a cloud resource.

  • Use Local Storage: Set whether to use local storage when the shared file system of the selected cloud resource does not have enough space or is unavailable in VM deployment.

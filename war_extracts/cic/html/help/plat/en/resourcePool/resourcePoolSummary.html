Display resource pool summary

Perform this task to display summary information about a resource pool.

Procedure

From the left navigation pane, select VDC > Resource Pools > Resource pool name.

Parameters

  • Basic Attributes: Basic attributes of the resource pool.

  • Cloud Resource: Cloud resource name.

  • Cloud Resource Type: Cloud resource type.

  • Description: Description of the resource pool.

  • CPUs: Number of physical CPUs in the resource pool.

  • Memory: Memory size of the resource pool.

  • Storage: Storage size of the resource pool.

  • Available Storage: Available storage size of the resource pool.

  • Compute Resource Monitoring Info: CPU usage and memory usage of the resource pool.

  • CPU: Host CPU usage dashboard graph.

  • Memory: Host memory usage dashboard graph.

  • Storage Resource Monitoring Info: Storage usage pie graph of the resource pool.

  • VM Allocation Info: Number of the resource pool's VMs that an organization uses and the proportion of the number of these VMs to the total number of VMs in the resource pool.

  • Organization: Organization that uses the resource pool.

  • VMs: Number of VMs used by the organization.

  • Ratio: Proportion of the number of the VMs used by the organization to the total number of VMs in the resource pool.

  • CPU Resource Allocation Info: Number of the resource pool's VM CPUs that an organization uses and the proportion of the number of these CPUs to the total number of CPUs in the resource pool.

  • Organization: Organization that uses the resource pool.

  • CPUs: Number of VM CPUs used by the organization.

  • Overcommit: Proportion of the number of the VM CPUs used by the organization to the total number of VMs in the resource pool.

  • Memory Resource Allocation Info: The resource pool's VM memory that an organization uses and the proportion of the used memory to the total memory in the resource pool.

  • Organization: Organization that uses the resource pool.

  • Memory: VM memory used by the organization.

  • Overcommit: Proportion of the VM memory used by the organization to the total memory in the resource pool.

Delete a resource pool

Perform this task to delete a resource pool.

Restrictions and guidelines

You cannot delete a resource pool that is being used by an organization. Before you delete a resource pool, remove the resource pool from the organization that is using the resource pool.

Procedure

  1. From the left navigation pane, select VDC > Resource Pools.

  1. Click the  icon in the Action column for a resource pool.

  1. In the dialog box that opens, click OK.

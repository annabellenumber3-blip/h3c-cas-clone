Display the task console

With the task console, operators can display information about tasks that have not been completed before login and tasks created by all operators after login, and the execution states of the tasks.

Procedure

  1. Click the  icon on the upper right corner of the page. The task console window will be displayed on the lower left corner.

  1. To close the task console, click the  icon.

Parameters

  • Task Name: Name of the task.

  • Operation Object: Target resource of the task.

  • Task State: Progress of task in a process bar with percentage.

  • Operator: Login name of the operator that created the task.

  • Operator IP: IP address of the computer that the operator used to create the task.

  • Started At: Time when the task started.

  • Completed At: Time when the task was completed.

  • Result: Execution result of the task.

  • Success: The task has been executed successfully.

  • Partial Success: The task has partially succeeded.

  • Failure: The task has failed to be executed.

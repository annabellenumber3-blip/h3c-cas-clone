Suspend a cloud host

Perform this task to suspend a running cloud host. When you suspend a cloud host, the system does not delete the data from the memory.

Restrictions and guidelines

You cannot suspend a vCenter cloud host.

Procedure

  1. From the left navigation pane, select Cloud Services > Cloud Hosts.

  1. Right-click the target cloud host and then select Suspend from the shortcut menu, or select the target cloud host, click More Operations, and then select Suspend.

  1. In the dialog box that opens, click OK.

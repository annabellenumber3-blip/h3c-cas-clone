Restart a cloud host

Perform this task to restart a running or suspended cloud host. You can restart a cloud host when it fails or has configuration changes, such as modifications to the number of CPUs, memory size, and disk size.

Procedure

  1. From the left navigation pane, select Cloud Services > Cloud Hosts.

  1. Right-click the target cloud host and then select Restart from the shortcut menu, or select the target cloud host, click More Operations, and then select Restart.

  1. In the dialog box that opens, click OK.

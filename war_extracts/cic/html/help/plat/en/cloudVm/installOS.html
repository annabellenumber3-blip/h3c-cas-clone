Install an OS on a cloud host

Perform this task to install an OS on a running cloud host after it is created.

Procedure

  1. From the left navigation pane, select Cloud Services > Cloud Hosts.

  1. Right-click the target cloud host and then select Console from the shortcut menu, or select the target cloud host, click More Operations, and then click Console.

  1. Click Virtual drive to load a local image file.

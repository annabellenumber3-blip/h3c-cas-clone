Manage cloud hosts

Perform this task to manage the VMs requested by users.

Restrictions and guidelines

The secure mode of a CVM takes effect only on cloud hosts of the CVM that are managed by CIC. The secure mode of CIC does not take effect on the cloud hosts.

Features

  • Display cloud hosts

  • Display user or user group information for a cloud host

  • Delete a user or user group for a cloud host

  • Start a cloud host

  • Suspend a cloud host

  • Resume a cloud host

  • Hibernate a cloud host

  • Restart a cloud host

  • Shut down a cloud host

  • Power off a cloud host

  • Delete a cloud host

  • Back up a cloud host immediately

  • Manage cloud host snapshots

  • Open the Web console of a cloud host

  • Manage a cloud host through the Web console

  • Install an OS on a cloud host

  • Assign a cloud host

  • Revoke a cloud host

Delete a user or user group for a cloud host

Perform this task to remove a user's or user group's access to a cloud host.

Procedure

  1. From the left navigation pane, select Cloud Services > Cloud Hosts.

  1. Click Show User/User Group.

  1. Select a cloud host.

  1. The User/User Group area displays the users and user groups that have access to the cloud host.

  1. Click the  icon in the Action column for a user or user group.

  1. In the dialog box that opens, click OK.

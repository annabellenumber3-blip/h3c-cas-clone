Hibernate a cloud host

Perform this task to hibernate a running or suspended cloud host. When a cloud host is hibernated, the data in its memory is saved as files in the disk, and the power of the cloud host is cut off.

Procedure

  1. From the left navigation pane, select Cloud Services > Cloud Hosts.

  1. Right-click the target cloud host and then select Hibernate from the shortcut menu, or select the target cloud host, click More Operations, and then select Hibernate.

  1. In the dialog box that opens, click OK.

Revoke a cloud host

Perform this task to revoke a cloud host that is already assigned to an organization, users, or user groups.

Procedure

  1. From the left navigation pane, select Cloud Services > Cloud Hosts.

  1. Right-click the target cloud host and select Revoke VM from the shortcut menu.

  1. Select an organization, users, or user groups and then click OK.

  1. In the dialog box that opens, click OK.

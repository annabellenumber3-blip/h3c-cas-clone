Assign a cloud host

Perform this task to assign a cloud host to users or user groups.

Procedure

  1. From the left navigation pane, select Cloud Services > Cloud Hosts.

  1. Right-click the target cloud host and select Assign VM from the shortcut menu, or select the target cloud host, click More Operations, and select Assign VM.

  1. Select a distribution method.

  • If you select User, click the  icon to select users. After you finish configuration, click OK.

  • If you select User Group, click the  icon to select user groups. After you finish configuration, click OK.

  1. Select a virtual desktop pool.

  1. Set the expiration date if you select the User method.

  1. Click OK.

Parameters

  • Organization: Organization that contains the users or user groups to be assigned the cloud host. This parameter is not configurable.

  • Distribution Method: Select the objects you want to assign the cloud host to.

  • User: Assign the cloud host to users. A user is a terminal user of the cloud host in the selected organization.

  • User Group: Assign the cloud host to user groups. A user group is a group of terminal users of the cloud host in the selected organization.

  • User: Select the users to be assigned the cloud host. This parameter is required. Click the  icon to select one or multiple users.

  • User Group: Select the user groups to be assigned the cloud host. This parameter is required. Click the  icon to select one or multiple user groups.

  • Virtual Desktop Pool: Select a virtual desktop pool from the virtual desktop pools of the organization.

  • Expire At: Select the expiration date of the cloud host. Users can use the cloud host only before this date. Set this parameter if you select the User method.

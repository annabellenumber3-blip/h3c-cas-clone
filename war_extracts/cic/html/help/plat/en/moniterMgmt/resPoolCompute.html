Display compute resource usage statistics

Perform this task to display compute resource usage statistics such as CPU and memory usages and save statistics reports.

Procedure

  1. From the left navigation pane, select Monitoring > Statistics Reports > Compute Resources.

  1. Select a host and statistics interval, set the start and end time, select an index, and then click Filter. The statistics will be displayed in the areas below.

  1. To display the statistics in a line chart or bar chart, click  and . To save the statistics as a picture, click .

  1. To clear the statistics, click Reset.

  1. To export the statistics, click Export.

Parameters

  • Host: Select a host. You can only select hosts that have been assigned to a resource pool.

  • Statistics Interval: Select a statistics collection interval. Options include Hour, Day, Week, Month, and Year.

  • From: Select the start time. It cannot be greater than the end time.

  • To: Select the end time. It cannot be smaller than the start time.

  • Index: Select an index. Options include CPU Usage, Memory Usage, Disk Usage, Disk Read Rate, Disk Write Rate, Disk I/O Throughput, Total Network Traffic, Total Network Read Traffic, Total Network Write Traffic, Total Network Read Rate, and Total Network Write Rate.

  • CPU Usage: CPU usage within the specified time period, including average value, max value, and min value.

  • Memory Usage: Memory usage within the specified time period, including average value, max value, and min value.

  • Disk Usage: Disk usage within the specified time period, including average value, max value, and min value.

  • Disk Read Rate Statistics (Mbps): Disk read rate within the specified time period, including average value, max value, and min value.

  • Disk Write Rate Statistics (Mbps): Disk write rate within the specified time period, including average value, max value, and min value.

  • I/O Throughput Statistics (Mb): Disk I/O throughput within the specified time period.

  • Total Network Traffic Statistics (Mb): Total network traffic within the specified time period.

  • Total Network Read Traffic Statistics (Mb): Total network read traffic within the specified time period.

  • Total Network Write Traffic Statistics (Mb): Total network write traffic within the specified time period.

  • NIC Read Rate Statistics (Mbps): Total network read rate within the specified time period, including average value, max value, and min value.

  • NIC Write Rate Statistics (Mbps): Total network write rate within the specified time period, including average value, max value, and min value.

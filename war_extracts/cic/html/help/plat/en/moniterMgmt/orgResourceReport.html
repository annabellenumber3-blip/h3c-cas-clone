Display organization resource quota usage statistics

Perform this task to display resource quota usages of an organization. Operators can set the filtering criteria such as organization, start time, and end time, and save the statistics.

Procedure

  1. From the left navigation pane, select Monitoring > Statistics Reports > Org. Resource Quotas. The list displays the organization resource quota usages over the last day.

  1. Click Advanced Filter.

  1. In the dialog box that opens, select an organization and the start time and end time, and then click OK.

  1. To print the data in the list, click Print.

  1. To export the data, click Export, and then select PDF or Excel.

Parameters

  • Organization: Select an organization.

  • From: Select the start time. It cannot be greater than the end time.

  • To: Select the end time. It cannot be greater than the end time.

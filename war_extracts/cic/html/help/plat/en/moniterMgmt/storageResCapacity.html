Display storage resource usage statistics

Perform this task to display storage resource usage statistics and save statistics reports.

Procedure

  1. From the left navigation pane, select Monitoring > Statistics Reports > Storage Resources.

  1. Select a storage pool and statistics interval, set the start and end time, and then click Filter. The statistics will be displayed in the areas below. If you do not select a storage pool, statistics reports for all shared storage pools that have been assigned to a resource pool will be displayed.

  1. To display the statistics in a line chart or bar chart, click the  icon and the  icon. To save the statistics as a picture, click the  icon.

  1. To clear the statistics, click Reset.

  1. To export the statistics, click Export.

Parameters

  • Storage Pool: Select one or multiple storage pools. You can select only storage pools that have been assigned to a resource pool. If you do not select any storage pools, storage pool usage statistics for all shared storage pools that have been assigned to a resource pool are displayed.

  • Statistics Interval: Select a statistics collection interval. Options include Hour, Day, Week, Month, and Year.

  • From: Select the start time. It cannot be greater than the end time.

  • To: Select the end time. It cannot be smaller than the start time.

  • Storage Usage: Storage usage within the specified time period.

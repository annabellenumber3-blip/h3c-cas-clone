Modify a customized monitoring panel

Perform this task to modify a customized monitoring panel.

Procedure

  1. From the left navigation pane, select Monitoring > Custom Monitoring.

  1. Click the  icon for the target monitoring panel.

  1. In the dialog box that opens, modify the monitoring panel name as needed, and then click OK.

  1. Click the target monitoring panel.

  1. Click Enter Design Mode.

  1. To delete a panel control, click the  icon in the upper right corner of the panel control. To modify a panel, click the  icon in the upper right corner of the panel control, and then modify the following settings as needed:

  • Health, host health, host state statistics, top 5 hosts by memory usage, top 5 hosts by CPU usage, VM state statistics, top 5 VMs by CPU usage, and top 5 VMs by memory usage.

  • Users, CPU statistics, memory statistics, and VMs.

  • CPU overcommitment, memory overcommitment, and shared storage overcommitment.

  1. Click Save, and then click Exit Design Mode.

Display VM usage details reports

Perform this task to display the startup and shutdown information within the specified time period for VMs assigned to an organization. Operators can set filtering criteria such as organization, user, VM name, VM startup time, and VM shutdown time, and save the statistics.

Procedure

  1. From the left navigation pane, select Monitoring > Statistics Reports > VM Usage Details. The list displays the start and shutdown statistics of VMs over the past day.

  1. Click Advanced Filter.

  1. In the dialog box that opens, select an organization, user or user group, enter the VM alias, select the start and shutdown time of the VM, and then click OK.

  1. To print the data in the list, click Print.

  1. To export the data, click Export, and then select PDF or Excel.

Parameters

  • Organization: Select an organization.

  • Alias: Enter the alias of the VM.

  • Type: Select user or user group.

  • User: User to which the VM is assigned.

  • User Group: User group to which the VM is assigned.

  • From: Select the startup time of the VM. It cannot be greater than the shutdown time of the VM. After you set this time, the start time of all the VMs in the list is later than this time.

  • To: Select the shutdown time of the VM. It cannot be smaller than the start time of the VM. After you set this time, the shutdown time of all the VMs in the list is earlier than this time.

Display VM usage summary reports

Perform this task to display startup and shutdown statistics for VMs assigned to an organization within the specified time period and save the statistics.

Procedure

  1. From the left navigation pane, select Monitoring > Statistics Reports > VM Usage Summary. The list displays the startup and shutdown statistics of VMs over the past day.

  1. Click Advanced Filter.

  1. In the dialog box that opens, enter the alias of the VM, select the start time and shutdown time of the VM, and then click OK.

  1. To display detailed startup and shutdown statistics of a VM, double-click the data for the target VM in the list.

  1. To print the data in the list, click Print.

  1. To export the data, click Export, and then select PDF or Excel.

Parameters

  • Alias: Alias of the VM.

  • From: Start time for filtering VMs by their startup time. It cannot be later than the end time. After you set the start time, the startup time of all the VMs in the list is later than this time.

  • To: End time for filtering VMs by their shutdown time. It cannot be earlier than the start time. After you set the end time, the shutdown time of all the VMs in the list is earlier than this time.

  • Start Time: Earliest startup time of the VM within the filtering time range.

  • Shutdown Time: Latest shutdown time of the VM within the filtering time range.

  • Up Time: Up time of the VM between the specified startup time and shutdown time.

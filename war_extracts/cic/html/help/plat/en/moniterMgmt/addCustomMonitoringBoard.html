Add a monitoring panel

Perform this task to add a monitoring panel.

Procedure

  1. From the left navigation pane, select Monitoring > Custom Monitoring.

  1. Click Add Monitoring Panel.

  1. In the dialog box that opens, enter the monitoring panel name, and then click OK.

  1. On the monitoring customization page, click the added monitoring panel.

  1. Click Enter Design Mode.

  1. Drag the target panel control in the panel control area to the blank area, and then configure the following settings:

  • For the health, host health, host state statistics, top 5 hosts by memory usage, top 5 hosts by CPU usage, VM state statistics, top 5 VMs by CPU usage, and top 5 VMs by memory usage panel controls, enter the panel title, select a monitored item (Organization or Resource Pool), select an organization or resource pool, and then click OK.

  • For the users, CPU statistics, memory statistics, and VMs panel controls, enter the panel title, select an organization ,and then click OK.

  • For the CPU overcommitment, memory overcommitment, and shared storage overcommitment panel controls, enter the panel title, select a resource pool, and then click OK.

  1. Click Save.

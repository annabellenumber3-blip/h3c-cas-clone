Display cloud resources

Perform this task to display all cloud resources in the system.

Procedure

From the left navigation pane, select Cloud Resources.

Parameters

  • Type: Type of the cloud resource.

  • IP Address: IP address of the cloud resource.

  • Username: Username for connecting to the cloud resource.

  • Action: Actions that operators can perform on the cloud resource.

  • To modify the cloud resource, click the  icon.

  • To delete the cloud resource, click the  icon.

  • To view details of the cloud resource, click the  icon.

  • To log in to the cloud resource, click the  icon.

  • To assign authority over the cloud resource, click the  icon.

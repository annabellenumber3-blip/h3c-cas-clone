Manage cloud resources

Cloud resources are the core nodes in the CAS layered management model. Cloud resource management manages all hardware infrastructures in data centers, including basic IT infrastructures (such as servers) and devices that serve the infrastructures (such as networks and the storage system). By default, CAS has a root node named Cloud Resources. You must add all physical resources under the root node for management.

The cloud resources managed by CIC include VMware vCenter cloud resources.

Features

  • Display cloud resources

  • Add a vCenter cloud resource

  • Modify a vCenter cloud resource

  • Delete a vCenter cloud resource

  • Display details of a vCenter cloud resource

  • Assign authority over a vCenter cloud resource

  • Manage vCenter cloud resources

  • Manage VM templates of a vCenter cloud resource

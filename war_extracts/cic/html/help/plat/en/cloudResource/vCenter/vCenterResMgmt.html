Manage vCenter cloud resources

vCenter cloud resource management allows you to manage the VM resources of the VMware vCenter platform. After you add VMware vCenter to CIC cloud resources, the hosts and VMs managed by VMware vCenter become part of CIC cloud resources and are centrally managed along by CIC.

You can view summary information for vCenter cloud resources and perform the following operations: start, hibernate, restart, shut down, power off, and delete VMs, create VM snapshots, clone VMs as VM templates, and assign VMs to users. You can assign the clusters and VM templates of a vCenter cloud resource to an organization. The administrator of the organization uses the VM templates to deploy new VMs in the cloud resource and to create the VMs requested by users.

Features

  • Display vCenter cloud resource summary

  • Display host information for a vCenter cloud resource

  • Display VM information for a vCenter cloud resource

  • Manually synchronize a vCenter cloud resource

  • Export VM information for a vCenter cloud resource

  • Manage VMs

Restart a VM

Perform this task to restart a VM in running state. You can restart a VM when it fails or has configuration changes, such as modifications to the number of CPUs, memory size, and disk size.

Procedure

  1. From the left navigation pane, select Cloud Resources > CVM cloud resource name.

  1. Click the VMs tab.

  1. Right-click the target VM and then select Restart from the shortcut menu.

  1. In the dialog box that opens, click OK.

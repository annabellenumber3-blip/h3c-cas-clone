Delete a VM

Perform this task to delete a VM.

Restrictions and guidelines

  • To avoid data loss, back up data before you delete VMs.

  • You can delete a VM in any state, including the running state.

  • When you delete a VM, the system automatically deletes all snapshots and hibernation information about the VM.

Procedure

  1. From the left navigation pane, select Cloud Resources > vCenter cloud resource name.

  1. Click the VMs tab.

  1. Right-click the target VM and then select Delete from the shortcut menu. In the dialog box that opens, select a deletion type and then click OK.

  1. In the dialog box that opens, click OK.

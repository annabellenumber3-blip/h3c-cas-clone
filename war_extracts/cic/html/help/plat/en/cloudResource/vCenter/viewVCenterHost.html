Display host information for a vCenter cloud resource

Perform this task to display host information for a vCenter cloud resource.

Procedure

  1. From the left navigation pane, select Cloud Resources > vCenter cloud resource name.

  1. Click the Hosts tab.

Parameters

  • Name: Host name.

  • Host Pool: Host pool where the host resides.

  • State: Host state.

  • VMs: Total number of VMs on the host, and the number of VMs in each state, including Running and Shutdown.

  • CPU Usage: Host CPU usage. This parameter is obtained from vCenter.

  • Memory Usage: Host memory usage. This parameter is obtained from vCenter.

  • Disk Size: Local disk size.

  • Platform: Virtualization platform used by the host.

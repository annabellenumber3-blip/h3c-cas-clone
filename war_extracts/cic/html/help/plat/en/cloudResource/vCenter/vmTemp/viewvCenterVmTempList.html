Display VM templates of a vCenter cloud resource

Perform this task to view all VM templates in a vCenter cloud resource and the network and storage information for a VM template.

Procedure

  1. From the left navigation pane, select Cloud Resources > vCenter cloud resource name.

  1. Click the Templates tab.

  1. Select a VM template, and click the Network tab to view network information about the VM template.

  1. Select the Storage tab to view the storage information for the VM template.

Parameters

Template

  • Name: VM template name.

  • CPUs: Number of CPU cores, which is the product of the number of CPUs and the number of cores per CPU.

  • Memory: Memory size.

  • Storage: Disk size.

  • OS: OS of the VM. The current software version supports only Windows and Linux.

  • Operation: Operations that operators can perform on the VM template. To distribute the VM template, click the  icon.

Network

  • MAC Address: MAC address of the VM used as the template source.

  • Network Name: Name of the virtual switch that the source VM uses.

  • Storage

  • Storage Volume Name: Storage file name.

  • Size: Disk size.

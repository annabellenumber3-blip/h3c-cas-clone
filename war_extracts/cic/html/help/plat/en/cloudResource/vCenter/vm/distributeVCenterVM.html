Assign a VM in a vCenter cloud resource to users or user groups

Perform this task to assign a VM in a vCenter cloud resource to users or user groups. If you assign a VM to a user group, all users in the user group can use the VM.

Procedure

  1. From the left navigation pane, select Cloud Resources > vCenter cloud resource name.

  1. Click the VMs tab.

  1. Right-click the target VM and then select Assign VM from the shortcut menu.

  1. Click the  icon to select an organization.

  1. Select a distribution method.

  • If you select User, click the  icon to select users. After you finish configuration, click OK.

  • If you select User Group, click the  icon to select user groups. After you finish configuration, click OK.

  1. Select a virtual desktop pool.

  1. Set the expiration date if you select the User method.

  1. Click OK.

Parameters

  • Organization: Select an organization.

  • Distribution Method: Select the objects you want to assign the VM to.

  • User: Assign the VM to users. A user is a terminal user of the VM in the selected organization.

  • User Group: Assign the VM to user groups. A user group is a group of terminal users of the VM in the selected organization.

  • User: Select the users to be assigned the VM. This parameter is required. Click the  icon to select one or multiple users.

  • User Group: Select the user groups to be assigned the VM. This parameter is required. Click the  icon to select one or multiple user groups.

  • Virtual Desktop Pool: Select a virtual desktop pool from the virtual desktop pools of the organization.

  • Expire At: Select the expiration date of the VM. Users can use the VM only before this date. Set this parameter if you select the User method.

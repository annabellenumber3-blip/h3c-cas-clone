Create a VM snapshot

Perform this task to create a snapshot for a VM.

Procedure

  1. From the left navigation pane, select Cloud Resources > vCenter cloud resource name.

  1. Click the VMs tab.

  1. Right-click the target VM and then select Create Snapshot from the shortcut menu.

  1. In the dialog box that opens, enter the snapshot name and description if the VM is in running state, specify whether to take memory snapshot and enable silent file system.

  1. Click OK.

Parameters

  • Memory Snapshot: Select whether to take memory snapshot. This parameter is displayed only when the VM is in running state.

  • Silent File System: Select whether to enable silent file system. For this parameter to take effect, you must install VMware Tools. This parameter is displayed only when the VM is in running state.

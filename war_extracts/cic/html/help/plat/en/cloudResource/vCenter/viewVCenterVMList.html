Display VM information for a vCenter cloud resource

Perform this task to display VM information for a vCenter cloud resource.

Procedure

  1. From the left navigation pane, select Cloud Resources > vCenter cloud resource name.

  1. Click the VMs tab.

Parameters

  • Name: VM name.

  • Host IP: IP address of the host where the VM resides.

  • State: State of the VM.

  • CPU Usage: VM CPU usage.

  • Memory Usage: VM memory usage.

  • vCPUs: Number of CPU cores, which is the product of the number of CPUs and the number of cores per CPU.

  • Memory: VM memory size.

  • Storage Size: VM disk size.

  • OS: OS of the VM. The current software version supports only Windows and Linux.

Display vCenter cloud resource summary

Perform this task to view summary configuration of a vCenter cloud resource.

Procedure

From the left navigation pane, select Cloud Resources > vCenter cloud resource name.

Parameters

  • Basic Attributes

  • Host Pool: Number of host pools.

  • Host: Number of hosts.

  • Total CPUs: Total number of CPUs on hosts.

  • Total Memory: Total size of memory on hosts.

  • Total Storage: Total size of local storage on hosts.

  • Local Available Storage: Total size of available local storage on hosts.

  • VM: Number of VMs in each state, including Running and Shutdown.

  • VM Distribution: Average number of VMs per host.

  • Host State Statistics: Total number of hosts in the organization, and the number of hosts in each state, including Normal, Maintain, and Abnormal.

  • Top 5 Hosts by CPU Usage: CPU usage trend graph for the top 5 hosts with the highest CPU usage.

  • Top 5 Hosts by Memory Usage: Memory usage trend graph for the top 5 hosts with the highest memory usage.

  • Top 5 VMs by CPU Usage: CPU usage trend graph for the top 5 VMs with the highest CPU usage.

  • Top 5 VMs by Memory Usage: Memory usage trend graph for the top 5 VMs with the highest memory usage.

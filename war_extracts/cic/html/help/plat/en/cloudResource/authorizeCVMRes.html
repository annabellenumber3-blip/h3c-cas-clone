Assign authority over a vCenter cloud resource

Perform this task to assign authority over a vCenter cloud resource to an operator group.

Procedure

  1. From the left navigation pane, select Cloud Resources.

  1. Click the  icon in the Action column for the target cloud resource.

  1. In the dialog box that opens, click the  icon.

  1. Select an operator group and then click OK.

  1. Click OK.

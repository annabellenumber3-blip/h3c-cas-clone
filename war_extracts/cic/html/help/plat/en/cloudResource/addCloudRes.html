Add a vCenter cloud resource

Perform this task to add a vCenter cloud resource.

Procedure

  1. From the left navigation pane, select Cloud Resources.

  1. Click Add Cloud Resource.

  1. In the dialog box that opens, specify the cloud resource type, resource name, description, IP address, protocol, port number, username, and password, and then click OK.

Parameters

  • IP Address: Enter the IP address of the cloud resource. The IP address must be in dotted-decimal format.

  • Protocol: Select the protocol used for logging in to the cloud resource. Options include HTTP and HTTPS. You must select HTTPS if the CIC login method is HTTPS.

  • Port Number: Enter the port number of the cloud resource. When you add a vCenter cloud resource, the default port number is 80 for HTTP and 443 for HTTPS.

  • Username: Enter the username for connecting to the cloud resource.

  • Password: Enter the password for connecting to the cloud resource.

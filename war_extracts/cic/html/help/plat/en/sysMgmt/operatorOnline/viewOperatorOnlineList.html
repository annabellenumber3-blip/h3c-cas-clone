Display online operators

Perform this task to display information about online operators in CIC.

Procedure

From the left navigation pane, select System > Operators & Groups > Online Operators.

Parameters

  • Login Name: Login name used by the operator to log in to CIC.

  • Operator: Name of the operator.

  • Operator Group: Operator group to which the operator belongs.

  • Login Time: Time when the operator logged in to CIC.

  • Login Address: IP address used by the operator for logging in to CIC.

  • Session ID: Session ID for this login.

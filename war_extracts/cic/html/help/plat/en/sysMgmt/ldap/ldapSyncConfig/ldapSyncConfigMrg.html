LDAP synchronization policy configuration

LDAP synchronization policy configuration allows you to synchronize users in a base DN of an LDAP server to an organization in CIC.

Restrictions and guidelines

Disabled users in the base DN of the LDAP server will also be synchronized to a CIC organization and displayed as valid, but they cannot log in to SSV.

Features

  • Display LDAP synchronization policies

  • Display LDAP synchronization policy details

  • Add an LDAP synchronization policy

  • Modify an LDAP synchronization policy

  • Delete an LDAP synchronization policy

  • Synchronize LDAP synchronization policies

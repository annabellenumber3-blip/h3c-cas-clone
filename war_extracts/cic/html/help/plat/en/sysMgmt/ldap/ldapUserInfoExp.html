Export LDAP users

Perform this task to export user attributes on an LDAP server.

Procedure

  1. From the left navigation pane, select System > LDAP User Sync > Export LDAP Users.

  1. In the dialog box that opens, select an LDAP server and specify a filtering criterion. Click Filter.

  1. Select an attribute, enter the export file name, select the file format and column separator, and enable or disable attribute name exportation.

  1. Click Export.

Parameters

  • LDAP Server: Select the LDAP server where the user to export is located.

  • Filter Criterion: Specify a filtering criterion.

  • Attribute Name: Select an attribute name. You must select a minimum of one attribute.

  • Sample Data: Sample data to help user view attribute names.

  • File Name: Enter the name of the file to save exported data.

  • File Format: Select a file format. Options include CSV and txt.

  • Delimiter: Select a delimiter for separating user attributes. If you select CSV as the file format, the delimiter is comma (,). If you select txt as the file format, options include Space, comma (,), semicolon (;), colon (;), TAB, pond sign (#), and dollar sign ($).

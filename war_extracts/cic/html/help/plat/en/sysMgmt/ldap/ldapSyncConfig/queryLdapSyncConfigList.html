Display LDAP synchronization policies

Perform this task to display LDAP synchronization policies.

Procedure

From the left navigation pane, select System > LDAP User Sync > Sync Policies.

Parameters

  • Sync Policy Name: Name of the LDAP synchronization policy.

  • LDAP Server: Name of the LDAP server used by the LDAP synchronization policy.

  • Organization: Name of the organization to which the user is synchronized.

  • User Group: Name of the user group to which the user is synchronized.

  • On-Demand Sync: Whether on-demand synchronization is enabled or not.

  • State: State of the LDAP synchronization policy.

  • Action: Actions that operators can perform on the LDAP synchronization policy.

  • To modify the LDAP synchronization policy, click the  icon.

  • To delete the LDAP synchronization policy, click the  icon.

  • To display the LDAP synchronization policy, click the  icon.

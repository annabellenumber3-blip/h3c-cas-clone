Add an LDAP server

Perform this task to add an LDAP server.

Procedure

  1. From the left navigation pane, select System > LDAP User Sync > LDAP Servers.

  1. Click Add LDAP Server.

  1. In the dialog box that opens, enter the server name, IP address, and port number, select a server type and version, set the connection timeout and synchronization timeout, and then click Next.

  1. Enter a base DN, admin DN, admin password, and username attribute. Click Connectivity Test.

  1. Click Finish.

Parameters

  • Server Name: Specify a name for the LDAP server.

  • IP Address: Specify an IP address for the LDAP server.

  • Port: Specify a port for connecting to the LDAP server.

  • Type: Select an LDAP server type. Options include Generic LDAP Server and Microsoft Active Directory.

  • Server Version: Specify a version for the LDAP server. Options include 2 and 3.

  • Connect Timeout: Specify the connection timeout.

  • Sync Timeout: Specify the synchronization timeout.

  • Base DN: Specify the absolute path of the directory that stores user data on the LDAP server. The base DNs of the LDAP servers cannot be duplicate. If the LDAP server is Microsoft Active Directory, specify the OU when configuring the base DN.

  • Admin DN: Specify the absolute path that locates the administrator on the LDAP server.

  • Admin Password: Enter a password for logging in to the LDAP server.

  • Username Attribute: Enter the attribute description used on the LDAP server for user names for logging in to SSV.

Display LDAP server information

Perform this task to display information about LDAP servers. All LDAP servers that the current operator can manage will be displayed on the list.

Procedure

From the left navigation pane, select System > LDAP User Sync > LDAP Servers.

Parameters

  • Server Name: Name of the LDAP server.

  • Type: Type of the LDAP server.

  • IP Address: IP address of the LDAP server.

  • Base DN: Absolute path of the directory that stores user data on the LDAP server.

  • Admin DN: Absolute path that locates the administrator on the LDAP server.

  • Action: Actions that operators can perform on the LDAP server.

  • To delete the LDAP server, click the  icon.

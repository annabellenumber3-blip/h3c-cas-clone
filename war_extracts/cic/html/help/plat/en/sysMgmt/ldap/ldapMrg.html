LDAP user synchronization configuration

An operator can use this feature to connect to an LDAP server and synchronize user information to CIC. After CIC establishes a connection with the LDAP server successfully, the operator must configure an LDAP synchronization policy to synchronize users in the specified base DN on the LDAP server to an organization by using different synchronization methods. In addition, CIC allows exportation of user attributes on an LDAP server.

Features

  • LDAP server configuration

  • LDAP synchronization policy configuration

  • Export LDAP users

Operator management

An operator is an administrator of CIC. For security purposes, an operator can use the functions of his own privilege only after the operator passes identity authentication. Operator management functions include add, modify, delete, search, and forbid operators, and display operator details. You can set privileges for an operator when you add or modify an operator.

Restrictions and guidelines

  • The system disables an operator's access to avoid malicious login attempt when the operator fails to provide a correct password within three attempts.

  • An admin operator is a root administrator, and has all privileges. An admin is automatically created when the CIC is installed, and cannot be deleted.

  • Access privileges of operators can be assigned through configuring an operator group or configured separately.

Features

  • Display operators

  • Add an operator

  • Modify an operator

  • Delete operators

  • Filter operators

  • Display detailed information about an operator

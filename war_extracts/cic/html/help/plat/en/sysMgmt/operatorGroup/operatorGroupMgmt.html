Operator group management

An operator group is a collection of operators. It can contain multiple levels of subgroups. Operators in an operator group or subgroup have the same access right. The default operator groups are system administrator group and organization administrator group.

By default, an operator in an operator group or sub group can display and operate only resources in his own group.

Features

  • Display operator groups

  • Add an operator group

  • Add a sub operator group

  • Modify an operator group or subgroup

  • Delete an operator group or subgroup

  • Display detailed information about an operator group

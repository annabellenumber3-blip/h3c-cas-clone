Display detailed information about an operator group

Perform this task to display detailed information about an operator group.

Procedure

  1. From the left navigation pane, select System > Operators & Groups > Operator Groups.

  1. For a parent group, click the  icon in the Action column. For a subgroup, click the  icon to the left of its parent group name, and then click the  icon in the Action column.

Parameters

  • Name: Name of the operator group.

  • Group Nesting: Whether the operator can manage all subgroups of the operator group to which the operator belongs.

  • Permission Settings: Actions that the operators in the operator group can perform.

Display operator groups

Perform this task to display information about operator groups in CIC, including parent groups and their subgroups. Operators can display subgroups of a parent group by clicking  before the parent group name.

Procedure

From the left navigation pane, select System > Operators & Groups > Operator Groups.

Parameters

  • Name: Name of the operator group.

  • Group Nesting: Whether an operator can manage all sub groups of the operator group to which the operator belongs.

  • Action: Actions that operators can perform on the operator group.

  • To add a subgroup, click the  icon.

  • To modify the user group, click the  icon.

  • To delete the user group, click the  icon.

  • To display user group information, click the  icon.

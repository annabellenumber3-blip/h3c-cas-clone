Add a sub operator group

Perform this task to add a subgroup for an operator group.

Procedure

  1. From the left navigation pane, select System > Operators & Groups > Operator Groups.

  1. For a parent group, click the  icon in the Action column. For a subgroup, click the  icon to the left of its parent group name, and then click the  icon in the Action column.

  1. In the dialog box that opens, enter the group name and description, and enable or disable group nesting as needed, and then click Next.

  1. Select the actions that the operators in the group can perform, and then click Finish.

Parameters

  • Name: Enter a name for the operator group. An operator group name must be unique in an organization.

  • Group Nesting: Enable or disable the operator to manage all subgroups of the operator group to which the operator belongs.

  • Permission Settings: Select the actions that the operators in the operator group can perform.

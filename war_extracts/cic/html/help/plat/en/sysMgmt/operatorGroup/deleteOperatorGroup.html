Delete an operator group or subgroup

Perform this task to delete an operator group or subgroup. An operator group that contains subgroups cannot be deleted.

Restrictions and guidelines

  • The default system admin group cannot be deleted.

  • To delete an operator group that contains subgroups, you must delete the subgroups first.

Procedure

  1. From the left navigation pane, select System > Operators & Groups > Operator Groups.

  1. For a parent group, click the  icon in the Action column. For a subgroup, click the  icon to the left of its parent group name, and then click the  icon in the Action column.

  1. In the dialog box that opens, click OK.

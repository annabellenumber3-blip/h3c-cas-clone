Display access policy details

Perform this task to display detailed information about an access policy.

Procedure

  1. From the left navigation pane, select System > Access Policies.

  1. Select the target access policy, and then click View.

Parameters

  • Default Access Type: Default access type when no access policy is matched.

  • Access Time Control: Time to allow access to the system.

  • Start IP: Start IP.

  • End IP: End IP.

  • Access Type: Access type.

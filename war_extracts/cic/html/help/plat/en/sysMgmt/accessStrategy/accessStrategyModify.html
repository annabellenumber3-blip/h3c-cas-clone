Modify an access policy

Perform this task to modify an access policy.

Restrictions and guidelines

The default access type and the access type specified in a rule cannot be the same.

Procedure

  1. From the left navigation pane, select System > Access Policies.

  1. Select the target access policy, and then click Modify.

  1. In the dialog box that opens, modify the policy name and description, select a default access type, or modify the access control time as needed.

  1. On the rule list, click  for the target rule.

  1. In the dialog box that opens, modify the start IP, end IP, and access type as needed, and then click OK.

  1. Click OK.

Parameters

  • Default Access Type: Select the default access type when no access policy is matched. Options include Allow and Deny.

  • Access Time Control: Select the time to allow access to the system.

  • Start IP: Enter the start IP.

  • End IP: Enter the end IP.

  • Access Type: Select an access type. Options include Allow and Deny.

Add an access policy

Perform this task to add an access policy.

Restrictions and guidelines

Make sure a minimum of one access type in a rule is different from the default access types when you add an access policy.

Procedure

  1. From the left navigation pane, select System > Access Policies.

  1. Click Add.

  1. In the dialog box that opens, enter the policy name and description, select a default access type, and set the access control time.

  1. Click Add Rule.

  1. In the dialog box that opens, enter the start IP and end IP, and select an access type.

  1. Click OK.

  1. Click OK.

Parameters

  • Default Access Type: Select the default access type when no access policy is matched. Options include Allow and Deny.

  • Access Time Control: Select the time to allow access to the system.

  • Start IP: Enter the start IP.

  • End IP: Enter the end IP.

  • Access Type: Select an access type. Options include Allow and Deny.

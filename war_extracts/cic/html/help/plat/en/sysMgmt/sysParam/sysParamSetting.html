Configure system parameters

Perform this task to configure system parameters such as basic parameters, customization parameters, and logout parameters.

Procedure

  1. From the left navigation pane, select System > Parameters.

  1. Click the System Params tab.

  1. Set basic system parameters: set the operator idle timeout, login failure lock timeout, maximum login attempts, and max number of online operators. Select a log level, monitoring interval, and log retaining duration and enable or disable the animation effect and security mode.

  1. Set customization parameters: enter the system name, Favicon, copyright information, copyright logo, login page background, login page logo, homepage background, and homepage logo.

  1. Set logout parameters: enter the VM logout delay and reminder lead time, select a logout action and deletion type. The deletion type parameter is required only when you select Delete VM as the logout action.

  1. Click Save.

Parameters

Basic System Params

  • Operator Idle Timeout: Select an idle timeout for the current webpage. The system automatically logs out an operator when the operator idle timeout expires.

  • Login Failure Lock Timeout: Specify the time during which an operator is locked when the maximum password attempts are reached.

  • Max. PWD Attempts: Specify the maximum number of password attempt failures. When the password attempts exceed this value, the operator will be disabled from logging in to the system for a time determined by the login failure lock timeout.

  • Max. Number of Online Operators: Enter the maximum number of concurrent online operators.

  • Log Level: Select a log level. Options include Error, Warning, and Tip.

  • Animation Effect: Enable or disable animation effect for the navigation tree.

  • Monitoring Interval: Select an interval for automatically monitoring host and VM performance. Options include 30 Seconds, 1 Minute, 5 Minutes, and 10 Minutes.

  • Secure Mode: Enable or disable secure mode. When secure mode is enabled, operator names and usernames cannot be modified, an operator account can be used by only one user at a time, and users can access CIC only through HTTPS.

  • Log Retaining Duration: Enter the log retaining duration. Operation logs generated within the log retaining duration cannot be deleted.

Customization Params

  • System Name: Specify a name for the CIC system.

  • Favicon: Set the path for the logo picture on the browser tab.

  • Copyright Info: Set the copyright information for the login page.

  • Copyright Logo: Specify a path for the copyright logo picture.

  • Login Page Background: Specify a path for the login page background picture.

  • Login Page Logo: Specify a path for the login page logo picture.

  • Homepage Background: Specify a path for the homepage background picture.

  • Homepage Logo: Specify a path for the homepage logo picture.

  • Display Columns Lines: Select whether to display lines between columns in a table.

Logout Params

  • VM Logout Delay (Days): Specify a time delay when the VM requested by the user expires. A value of 0 indicates that the VM will expire immediately when the expire time is reached.

  • Logout Action: Select the action to take when the VM expires. Options include Remove Privilege and Delete VM.

  • Remove Privilege: When a VM expires, users cannot use the VM, but the VM still exists in CIC.

  • Delete VM: When a VM expires, the VM is deleted from CIC.

  • Deletion Type: Specify a deletion type. Options include Save Image and Delete Image. This parameter is required if you select Delete VM as the logout action.

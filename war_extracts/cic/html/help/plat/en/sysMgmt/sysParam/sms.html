Configure SMS settings

Perform this task to configure SMS settings. The system automatically sends notification messages if SMS notification is enabled in an approval workflow.

Procedure

  1. From the left navigation pane, select System > Parameters.

  1. Click the SMS Configuration tab.

  1. Enable or disable SMS notification, and then click Save.

Parameters

  • SMS Notification: Enable or disable SMS notification.

  • SMS Type: Select a platform for sending short messages. Options include HCTCOM SMS Platform and Huawei SMS Platform.

  • HCTCOM SMS Platform: A third-party SMS platform. It requires registration and deployment. For more information, see the HCTCOM website.

  • Huawei SMS Platform: Huawei SMS platform.

  • Login Name: Enter a login name for logging in to the HCTCOM SMS platform or Huawei SMS platform.

  • Login Password: Enter a password for logging in to the HCTCOM SMS platform or Huawei SMS platform.

  • HTTP Proxy: Enable or disable HTTP proxy.

  • Proxy Address: Enter the IP address of the HTTP proxy.

  • Proxy Port: Enter the port of the HTTP proxy.

  • Proxy Username: Enter the username of the HTTP proxy.

  • Proxy Password: Enter the password of the HTTP proxy.

  • IP Address: Enter the IP address of the Huawei SMS platform.

  • Database Username: Enter the username used for logging in to the Huawei SMS database.

  • Password: Enter the password used for logging in to the Huawei SMS database.

  • Caller Number Identification: Enter the caller number identification of the Huawei SMS platform.

  • Service Code: Enter the service code of the Huawei SMS platform.

  • Services: Select the services that require the system to send short messages to notify the operators.

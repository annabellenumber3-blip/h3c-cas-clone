Mail server configuration

Perform this task to configure SMTP mail server settings.

Procedure

  1. From the left navigation pane, select System > Parameters.

  1. Click the Mail Server tab.

  1. Specify the IP address and port number of the mail server, and enable or disable identity authentication and SSL encryption as needed. If you enable identity authentication, enter a username and password for logging in to the mail server.

  1. Enter the sender's email address and name.

  1. Click Save.

Parameters

  • Server Address: Enter the IP address or domain name of the SMTP mail server.

  • Port Number: Enter the TCP port number for connecting to the mail server.

  • Identity Authentication: Enable or disable identity authentication on the sender.

  • SSL Encryption: Enable or disable SSL encryption.

  • Username: Specify an account for the mail server. This parameter is required if you select Identity Authentication.

  • Password: Enter the password for logging in to the mail server. The parameter is required if you select Identity Authentication.

  • Sender Name: Enter the name of the user that has sent the email.

  • Sender's Email Address: Enter the email address of the sender.

Configure a certificate

Perform this task to configure a certificate so that operators can log in to the system through two-factor authentication. After two-factor authentication is enabled, operators can log in to the system only through USB keys.

Restrictions and guidelines

For successful login through USB keys, make sure the uploaded certificate is correct and the USB keys are configured correctly when you configure two-factor authentication.

Procedure

  1. From the left navigation pane, select System > Certificates.

  1. Enable two-factor authentication, set certificate parameters, and then click Save.

Parameters

  • Enable 2FA: After you enable two-factor authentication, operators log in to the system through USB keys instead of usernames and passwords.

  • Root Certificate: Select a root certificate file for verifying the USB key certificate. The size of the root certificate file cannot exceed 5 MB. The selected root certificate file will be automatically uploaded to the system.

  • Update URL: Enable or disable scheduled URL update.

  • Link: Enter the link for searching all revoked certificates.

  • Frequency: Select the frequency at which URLs are updated. Options include Monthly, Weekly, and Daily. If you select Weekly, select a day. If you select Monthly, select a date.

  • Time: Select the time at which URLs are updated.

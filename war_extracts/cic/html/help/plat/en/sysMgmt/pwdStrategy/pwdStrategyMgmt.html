Password policy management

A password policy manages password complexity and password expiration. It takes effect on all the operators.

Restrictions and guidelines

A password policy takes effect immediately after being modified. The newly registered operators and the operators modifying the password must comply with the strategy. The operators that have registered are not affected.

Procedure

  1. From the left navigation pane, select System > Password Policy.

  1. Enter the minimum password length and valid period, and select a complexity requirement.

  1. Click Save.

Parameters

  • Min. Length: Enter the minimum length of the password.

  • Complexity: Select password complexity requirement. Options include No Requirement, Must Contain Letters and Numbers, Must Contain Special Characters, Must Contain Letters, Numbers, and Special Characters, and Must Contain Upper-Case Letters, Lower-Case Letters, Numbers, and Special Characters. For more information about special characters, see the table below.

  • Validity Period: Enter the validity period of the password.

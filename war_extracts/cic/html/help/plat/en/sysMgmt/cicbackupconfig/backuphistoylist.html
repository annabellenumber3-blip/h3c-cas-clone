Display backup history

Perform this task to display backup history for all CIC data and configuration files that have been automatically and manually backed up.

Procedure

  1. From the left navigation pane, select System > CIC Backup.

  1. Click the Backup History tab.

Parameters

  • File Name: Name of the backup file.

  • Created At: Time when the backup file was created.

  • Version: CIC version number when the backup was executed.

  • Destination: Select a backup destination.

  • IP Address: IP address of the remote server.

  • Connection Type: Type of connection to the remote server This parameter is displayed if you select Remote Server as the backup destination.

  • Location: Directory on the remote server to save the backup file.

  • Action: Actions that operators can perform on the backup file.

  • To restore the CIC data, click the  icon.

  • To delete the backup file, click the  icon.

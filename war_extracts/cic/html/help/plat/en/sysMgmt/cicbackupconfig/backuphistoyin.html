Import backup files

Perform this task to import backup files.

Procedure

  1. From the left navigation pane, select System > CIC Backup.

  1. Click the Backup History tab.

  1. Click Import.

  1. In the dialog box that opens, select the backup destination. Options include Local Host and Remote Server.

  • If you select Local Host, click the  icon to select a backup location, or enter a backup location.

  • If you select Remote Server, specify an IP address, username, password, and backup location, select a connection type, and specify a service port, and then click Connectivity Test. Enter a backup location.

  1. Click OK.

Parameters

  • Destination: Select a backup destination. Options include Local Host and Remote Server.

  • Location: If you select Local Host as the backup destination, this parameter indicates the directory on the local host where the CIC backup files are saved. If you select Remote Server as the backup destination, this parameter indicates the directory on the remote server where the CIC backup files are saved.

  • IP Address: Specify the IP address of the remote server. This field is required if you select Remote Server as the backup destination.

  • Username: Specify an account for the remote server. This account must have read and writes rights. If you select FTP as the connection type, this account must be an account created on the remote FTP server. If you select SCP, this account must be an account created on the OS of the remote server. This field is required if you select Remote Server as the backup destination.

  • Password: Enter the password used for connecting to the remote server. This field is required if you select Remote Server as the backup destination.

  • Connection Type: Specify a connection type for connecting to the remote server. Options include FTP and SCP. This field is required if you select Remote Server as the backup destination.

  • Server Port: Specify a port for accessing the remote server. This field is required if you select Remote Server as the backup destination.

Clear operation logs

Perform this task to clear operation logs earlier than the specified time. Operators can set the log retaining duration in the Basic System Params area on the System > Parameters page.

Procedure

  1. From the left navigation pane, select System > Operation Logs.

  1. Click Clear Operation Logs.

  1. In the dialog box that opens, click on the Clear Logs Before field, or click  to select a start time from the calendar dialog box that opens.

  1. Click OK.

Parameters

  • Clear Logs Before: Select a date before which the operation logs will be cleared. The default date is three months before the current date. For example, if the current date is October, 24th, 2014, operation logs before July, 24th, 2014 will be cleared by default. If use a custom date, make sure the date is earlier than the default date.

System management

System management allows you to view and configure CIC system parameters.

System management provides the following features:

  • Operator and operator group management: Manages operators, operator groups, and online operators.

  • Password policy management Manages passwords and password loss handling methods.

  • Access policy management: Controls operator access by using ACLs.

  • Configure a certificate: Manages system login through certificates.

  • CIC backup management: Backs up and restores CIC data and configuration files.

  • LDAP user synchronization configuration: Authenticates users and synchronizes user information to an organization by configuring an LDAP server.

  • Operation log management Filters operator and user operation logs.

  • Collect log files: Collects CIC system logs.

  • System parameter configuration: Configures system parameters and mail servers.

  • License management: Provides license request and registration.

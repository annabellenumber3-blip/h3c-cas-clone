Request a license

Perform this task to obtain a host information file, which defines how CIC components are licensed.

Procedure

  1. From the left navigation pane, select System > Licenses.

  1. Click Request.

  1. In the dialog box that opens, enter basic information about the terminal user, such as username, province/city, company/organization, select the country/region, and then click Next.

  1. Enter the name, company/organization, email address, and contact of the applicant.

  1. Click Finish. The browser will automatically download a host file, or ask you to save the host file. The default name of the host file is host.info.

Parameters

Terminal User Info

  • Username: Enter the full name of the terminal user that uses CIC.

  • Country/Region: Select the country or region where the terminal user is located.

  • Province/City: Enter the province or city where the terminal user is located.

  • Company/Org: Enter the full name of the company or organization where the terminal user is located.

  • Address: Enter the contact address of the company or organization to which the terminal user belongs.

  • Postcode: Enter the postcode of district where the company or organization to which the terminal user belongs is located.

  • Email Address: Enter the email address of the terminal user.

  • Phone Number: Enter the phone number of the terminal user.

Applicant Profile

  • Name: Enter the full name of the applicant.

  • Company/Org: Enter the full name of the company or organization where the applicant is located.

  • Email Address: Enter the email address of the applicant.

  • Phone Number: Enter the phone number of the applicant.

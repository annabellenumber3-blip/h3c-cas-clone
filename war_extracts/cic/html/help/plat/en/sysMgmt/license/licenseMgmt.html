License management

CIC licenses include licenses for the CIC component and licenses for the Virtual Desktop Infrastructure (VDI) component. One CIC component license manages one VM in CIC. One VDI component license manages one user in CIC.

CIC provides a 180-day trial version. A trial version provides the same functions as a formal version. You can contact an authorized distributor to apply for a formal version.

Features

  • Display license details

  • Request a license

  • Register a license

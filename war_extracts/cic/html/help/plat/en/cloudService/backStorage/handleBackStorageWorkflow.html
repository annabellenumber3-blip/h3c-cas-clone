Process a cloud backup policy workflow

Perform this task to process a cloud backup policy workflow.

Procedure

  1. From the left navigation pane, select Workflows > Backup Policy Tasks.

  1. Select To Be Processed from the State list, and then click the   icon in the Action column for the target cloud backup policy workflow.

  1. Select a processing result and enter the comment. If you select Transfer, specify an operator.

  1. Click OK.

Parameters

Workflow Info

  • Type: Type of the workflow.

  • Username: Username of the terminal user that submitted the workflow.

  • Cloud Backup Policy Name: Name of the cloud backup policy that the terminal user has requested.

  • Effective Period: Time period during which the cloud backup policy is valid.

  • VM: VM that the terminal user has requested to back up.

Approval Info

  • Operator: Operator that has processed the workflow.

  • Result: Processing result of the workflow.

  • Comment: Comment on the workflow.

  • Processed At: Time when the operator processed the workflow.

Processing Info

  • Result: Select the processing result of the workflow. Options include Approve, Refuse, and Transfer.

  • Operator: Select an operator if you select Transfer.

  • Comment: Enter the comment on the workflow.

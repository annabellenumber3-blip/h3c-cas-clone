Display cloud resource anti-virus configuration details

Perform this task to view details about the anti-virus patch installed on CVM and the CVK hosts managed by CVM.

Procedure

  1. From the left navigation pane, select Cloud Services > Cloud Security > Anti-Virus.

  1. Click the  icon for a cloud resource.

Parameters

  • Cloud Resource Name: Cloud resource name.

  • Antivirus Vendor: Vendor of the anti-virus patch.

  • Patch Directory: Directory of the anti-virus patch.

  • Patch Name: Name of the anti-virus patch.

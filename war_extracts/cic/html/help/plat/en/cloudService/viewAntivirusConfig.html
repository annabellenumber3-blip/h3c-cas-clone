Display anti-virus configuration

Perform this task to view whether anti-virus software is installed on CVM and the CVK hosts managed by CVM.

Procedure

From the left navigation pane, select Cloud Services > Cloud Security > Anti-Virus.

Parameters

  • Cloud Resource Name: Cloud resource name.

  • Host State: Host state.

  • IP Address: IP address of the host.

  • Patch Installation State: Whether the anti-virus patch is installed on the host.

  • Anti-Virus Vendor: Vendor of the anti-virus patch.

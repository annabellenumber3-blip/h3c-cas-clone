Display user pre-registration workflow processing records

Perform this task to display pre-registration workflow processing records.

Procedure

  1. From the left navigation pane, select Workflows > Preregistration Tasks.

  1. Click the  icon in the Action column for the target pre-registration workflow.

Parameters

Workflow Info

  • Login Name: Login name registered by the terminal user.

  • Username: Username registered by the terminal user.

  • Email: Email address registered by the terminal user.

  • Phone Number: Phone number registered by the terminal user.

  • Identity No.: Identify number registered by the terminal user.

  • Contact: Contact address registered by the terminal user.

Approval Info

  • Operator: Operator that has processed the workflow.

  • Result: Processing result of the workflow.

  • Comment: Comment on the workflow.

  • Processed At: Time when the operator processed the workflow.

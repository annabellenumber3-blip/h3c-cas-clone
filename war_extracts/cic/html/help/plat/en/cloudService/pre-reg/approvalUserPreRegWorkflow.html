Implement a pre-registration workflow

Perform this task to implement a pre-registration workflow submitted by a terminal user.

Procedure

  1. From the left navigation pane, select Workflows > Preregistration Tasks.

  1. Click the  icon in the Action column for the target pre-registration workflow.

  1. In the dialog box that opens, select an organization and user group, and then click OK.

Parameters

  • Login Name: Login name registered by the terminal user for logging in to SSV.

  • Username: Name registered by the terminal user.

  • Identity No.: Identify number registered by the terminal user.

  • Email: Email address registered by the terminal user.

  • Phone Number: Phone number registered by the terminal user.

  • Contact: Contact address registered by the terminal user.

  • Organization: Organization of the terminal user.

  • User Group: User group of the terminal user.

Display user pre-registration workflows

Perform this task to display user pre-registration workflow summary.

Procedure

From the left navigation pane, select Workflows > Preregistration Tasks.

Parameters

  • Login Name: Login name that the terminal user uses to log in to CAS SSV.

  • Username: Name of the terminal user.

  • State: State of the workflow.

  • Identity No.: Identify number of the terminal user.

  • Email: Email address of the terminal user.

  • Phone Number: Phone number of the terminal user.

  • Contact: Contact address of the terminal user.

  • Action: Actions that operators can perform on the pre-registration workflow.

  • To process the pre-registration workflow, click the  icon.

  • To display the pre-registration workflow, click the  icon.

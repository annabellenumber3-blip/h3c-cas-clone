Process a pre-registration workflow

Perform this task to process a pre-registration workflow submitted by a terminal user.

Procedure

  1. From the left navigation pane, select Workflows > Preregistration Tasks.

  1. Select To Be Processed from the State list, and then click the  icon in the Action column for the target pre-registration workflow.

  1. Select a processing result and enter the comment. If you select Transfer, specify an operator.

  1. Click OK.

Parameters

Workflow Info

  • Login Name: Login name requested by the user in the pre-registration workflow.

  • Username: Username requested by the user in the pre-registration workflow.

  • Email: Email address of the user that submitted the pre-registration workflow.

  • Phone Number: Phone number of the user that submitted the pre-registration workflow.

  • Identity Number: Identity number of the user that submitted the pre-registration workflow.

  • Contact: Contact address of the user that submitted the pre-registration workflow.

Approval Info

  • Operator: Operator that has processed the workflow.

  • Result: Processing result of the workflow.

  • Comment: Comment on the workflow.

  • Processed At: Time when the operator processed the workflow.

Processing Info

  • Result: Select the processing result on the workflow. Options include Approve, Refuse, and Transfer.

  • Operator: Select an operator if you select Transfer.

  • Comment: Enter the comment on the workflow.

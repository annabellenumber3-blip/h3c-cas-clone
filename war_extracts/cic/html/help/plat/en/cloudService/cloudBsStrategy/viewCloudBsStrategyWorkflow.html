Display cloud backup policy workflows

Perform this task to display cloud backup policy workflow summary.

Procedure

From the left navigation pane, select Workflows > Backup Policy Tasks.

Parameters

  • Cloud Backup Policy Name: Name of the cloud backup policy that the terminal user has requested.

  • Username: Name of the terminal user.

  • Effective Period: Time period during which the cloud backup policy is valid.

  • State: State of the cloud backup policy workflow.

  • Action: Actions that operators can perform on the cloud backup policy workflow.

  • To process the cloud backup policy workflow, click the  icon.

  • To display the cloud backup policy workflow, click the  icon.

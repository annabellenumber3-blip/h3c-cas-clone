Implement a cloud backup policy workflow

Perform this task to implement a cloud backup policy workflow.

Procedure

  1. From the left navigation pane, select Workflows > Backup Policy Tasks.

  1. Click the  icon in the Action column for the target cloud host workflow.

  1. In the dialog box that opens, enter the number of backups to save and the comment, and then click OK.

Parameters

  • Cloud Backup Policy Name: Name of the cloud backup policy that the terminal user has requested.

  • Effective Period: Time at which the backup policy takes effect.

  • VM: VM that the terminal user has requested to back up.

  • Comment: Enter the comment on the workflow.

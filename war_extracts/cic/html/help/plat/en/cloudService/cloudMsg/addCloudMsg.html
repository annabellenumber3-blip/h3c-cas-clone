Add a cloud message

Perform this task to add a cloud message and send the message to a user or user group.

Procedure

  1. From the left navigation pane, select Workflows > Cloud Messages.

  1. Click Add.

  1. In the dialog box that opens, click on the Target User field, or click the  icon to the right of this field.

  1. Select a user from the Users Not Receiving This Message field, and then click Add.

  1. Click OK.

  1. Click on the Target Group field, or click the  icon to the right of this field.

  1. Select a user group.

  1. Click OK.

  1. Enter the title and content, and select a message level, and then click Send.

Parameters

  • Target User: Select a user to receive the cloud message.

  • Target Group: Select a user group to receive the cloud message.

  • Title: Enter the title of the cloud message.

  • Message Level: Select the severity level of the message. Options include Notification and Important.

  • Content: Enter the content of the cloud message.

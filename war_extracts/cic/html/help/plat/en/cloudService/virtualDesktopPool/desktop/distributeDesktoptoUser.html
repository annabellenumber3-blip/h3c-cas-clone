Assign a fixed virtual desktop

Perform this task to assign a fixed virtual desktop to a user or user group.

  • For a fixed virtual desktop pool, you can manually assign its virtual desktops to users or user groups.

  • A floating virtual desktop pool does not support manual desktop assignment. When a user in the user group assigned a floating desktop pool logs in to SSV, the system automatically assigns a random virtual desktop to the user.

Procedure

  1. From the left navigation pane, select VDC > Virtual Desktop Pools > virtual desktop pool name.

  1. Click the Virtual Desktops tab.

  1. Right-click the target virtual desktop and then select Assign VM from the shortcut menu.

  1. Select the objects you want to assign the virtual desktop to.

  • If you select User, click the  icon to select users. Click Add to add users to the list for users to be assigned the VM. Click OK.

  • If you select User Group, click the  icon to select user groups.

  1. Select a virtual desktop pool.

  1. Set the expiration date if you select the User method.

  1. In the dialog box that opens, click OK.

Parameters

  • Organization: Organization name.

  • Distribution Method: Select the objects you want to assign the virtual desktop to.

  • User: Assign the virtual desktop to users.

  • User Group: Assign the virtual desktop to user groups.

  • User: Select the users to be assigned the virtual desktop. This parameter is required. Click the  icon to select one or multiple users.

  • User Group: Select the user groups to be assigned the virtual desktop. This parameter is required. Click the  icon to select one or multiple user groups.

  • Virtual Desktop Pool: Select a virtual desktop pool from the virtual desktop pools of the organization.

  • Expire At: Select the expiration date of the virtual desktop. Users can use the virtual desktop only before this date.

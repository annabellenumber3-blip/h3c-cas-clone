Modify a virtual desktop pool

Perform this task to modify the configuration of a virtual desktop pool.

Restrictions and guidelines

  • The number of virtual desktops you set cannot be smaller than the number of virtual desktops that are already in the desktop pool.

  • You cannot modify the organization or resource pool for a desktop pool that already contains virtual desktops.

Procedure

  1. From the left navigation pane, select VDC > Virtual Desktop Pools.

  1. Click the  icon in the Action column for a virtual desktop pool.

  1. Enter the virtual desktop pool description and the number of desktops, select whether to deploy VMs after desktop pool creation if you are creating a floating desktop pool, and then click Next.

  1. Select an organization, a VM template, a resource pool, and a storage. If you are creating a floating desktop pool, select a user group. Click Next.

  1. If you use a Windows VM template, select a color depth, and specify whether to enable the local disk mapping, clipboard mapping, printer mapping, USB mapping, serial port mapping, and local audio mapping.

  1. Click Finish.

Parameters

  • Desktops: Enter the number of virtual desktops in the desktop pool.

  • For a fixed desktop pool, this parameter is the maximum number of assignable virtual desktops.

  • For a floating desktop pool, this parameter is the number of assigned virtual desktops.

  • Assignment Mode: Select the mode of virtual desktop assignment.

  • Fixed Desktop Pool: Virtual desktops in the desktop pool are manually assigned to users by an administrator. A user can use only the desktop assigned to him before the expiration time.

  • Floating Desktop Pool: Virtual desktops are automatically assigned to users by the system. When a user in the user group associated with a floating desktop pool logs in to SSV, the system automatically assigns a virtual desktop to the user. When the user logs out of SSV, the system automatically reclaims the virtual desktop.

  • Organization: Select an organization.

  • VM Template: Select a VM template. Options are VM templates owned by the selected organization.

  • Resource Pool: Select a resource pool. Options are resource pools that belong to the selected organization.

  • Storage: Select a storage pool. Options are storage pools that belong to the selected organization.

  • User Group: Select a user group. Options are user groups of the selected organization. Set this parameter when you are creating a floating desktop pool.

  • Color Depth: Select a color depth for remote sessions. Options include Best Quality (32 Bits), True Color (24 Bits), and High Color (16 Bits). This parameter is required only if you select a VM template installed with a Windows OS.

  • Local Disk: Set whether to allow users to use local disks. This parameter is required only if you select a VM template installed with a Windows OS.

  • Clipboard: Set whether to allow copy and paste operations between a client and a server. This parameter is required only if you select a VM template installed with a Windows OS.

  • Printer: Set whether to allow users to use local printers. This parameter is required only if you select a VM template installed with a Windows OS.

  • USB: Set whether to allow users to use local USB devices. This parameter is required only if you select a VM template installed with a Windows OS.

  • Serial Port: Set whether to allow users to use local serial port devices. This parameter is required only if you select a VM template installed with a Windows OS.

  • Local Audio: Set whether to allow users to use local audio devices. This parameter is required only if you select a VM template installed with a Windows OS.

  • Deploy Now: Set whether to deploy VMs immediately after the virtual desktop pool is created. Set this parameter when you are creating a floating desktop pool.

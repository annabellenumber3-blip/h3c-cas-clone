Manage virtual desktops of a virtual desktop pool

Perform this task to manage the virtual desktops (or VMs) in a virtual desktop pool. You can perform the following operations: start, suspend, resume, hibernate, restart, shut down, power off, assign, and reclaim VMs, manage snapshots, backup VMs, delete VMs, access the console, and view user or user group information for virtual desktops. For more information about basic virtual desktop management features, see VM management.

Features

  • Display virtual desktop information for a virtual desktop pool

  • Display user or user group information for a virtual desktop

  • Assign a fixed virtual desktop

  • Revoke a virtual desktop

  • Start a virtual desktop

  • Suspend a virtual desktop

  • Resume a virtual desktop

  • Hibernate a virtual desktop

  • Restart a virtual desktop

  • Shut down a virtual desktop

  • Power off a virtual desktop

  • Delete a virtual desktop

  • Reclaim a virtual desktop

  • Back up a virtual desktop immediately

  • Manage virtual desktop snapshots

  • Open the Web console of a virtual desktop

  • Manage a virtual desktop through the Web console

  • Install an OS on a virtual desktop

Display VM template information for a virtual desktop pool

Perform this task to display all VM templates of a virtual desktop pool.

Procedure

  1. From the left navigation pane, select VDC > Virtual Desktop Pools > virtual desktop pool name.

  1. Click the VM Templates tab.

Parameters

  • CPUs: Number of virtual CPUs.

  • Memory: Memory size.

  • Storage: Storage size.

  • Created At: Time when the VM template was created.

  • Action: Actions that operators can perform on the VM template.

  • To deploy a virtual desktop, click the  icon.

  • To bulk deploy virtual desktops, click the  icon.

Display network and storage information for a VM template

Perform this task to display network and storage information for a VM template.

Procedure

  1. From the left navigation pane, select VDC > Virtual Desktop Pools > virtual desktop pool name.

  1. Click the VM Templates tab.

  1. Select a VM template. The Network tab displays network information about the VM template. Click the Storage tab to view its storage information.

Parameters

  • Network

  • MAC Address: MAC address of the NIC.

  • vSwitch: Virtual switch name.

  • Profile: Port profile name.

  • VLAN: VLAN ID.

  • Avg Outbound Bandwidth: Limit on the average bandwidth of the traffic from the virtual switch to the VM.

  • Outbound Burst Buffer: Burst buffer for the traffic from the virtual switch to the VM.

  • Avg Inbound Bandwidth: Limit on the average bandwidth of the traffic from the VM to the virtual switch.

  • Inbound Burst Buffer: Burst buffer for the traffic from the virtual switch to the VM.

  • Storage

  • Device Name: Storage device name.

  • Bus Type: Bus type of the storage device.

  • Storage Volume Name: Storage file name.

  • Size: Disk size.

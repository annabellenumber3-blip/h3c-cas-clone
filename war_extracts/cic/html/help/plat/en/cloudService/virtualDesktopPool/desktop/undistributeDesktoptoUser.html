Revoke a virtual desktop

Perform this task to revoke a fixed virtual desktop that is assigned to an organization, users, or user groups. You cannot revoke a floating virtual desktop.

Procedure

  1. From the left navigation pane, select VDC > Virtual Desktop Pools > virtual desktop pool name.

  1. Click the Virtual Desktops tab.

  1. Right-click the target virtual desktop and then select Revoke VM from the shortcut menu.

  1. Select an organization, users, or user groups and click OK.

  1. In the confirmation dialog box, click OK.

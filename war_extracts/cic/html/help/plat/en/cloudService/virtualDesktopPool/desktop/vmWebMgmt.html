Manage a virtual desktop through the Web console

Perform this task to manage a virtual desktop through the Web console. You can perform the following operations through the Web console: Access the virtual desktop, refresh the console interface, send key combinations to the console, attach the virtual optical drive, disconnect or connect the virtual desktop, and close the console.

Procedure

  1. From the left navigation pane, select VDC > Virtual Desktop Pools > virtual desktop pool name.

  1. Click the Virtual Desktops tab.

  1. Right-click the target virtual desktop and then select Console from the shortcut menu.

  • To start or shut down the virtual desktop, click Start or Shut down.

  • To power off the virtual desktop, click Power off.

  • To refresh the console interface, click Refresh.

  • To send key combinations to the console, click Send.

  • To load an image file, click Virtual drive. You can select an ISO file from the local host or a remote shared directory.

  • To disconnect or connect the virtual desktop, click Disconnect or Connect.

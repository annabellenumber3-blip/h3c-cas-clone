Manage virtual desktop pools

Perform this task to bulk deploy virtual desktops and assign a virtual desktop to a certain organization. Users in an organization can access virtual desktops through a remote desktop protocol.

Features

  • Display virtual desktop pools

  • Add a virtual desktop pool

  • Modify a virtual desktop pool

  • Delete a virtual desktop pool

  • Configure a reclamation policy

  • Perform a health test

  • Display virtual desktop pool summary

  • Manage virtual desktops of a virtual desktop pool

  • Manage VM templates of a virtual desktop pool

Configure a reclamation policy

Perform this task to configure the reclamation policy for a virtual desktop pool. CIC automatically reclaims desktops for a floating desktop pool when the reclamation requirements of the active reclamation policy are met.

Procedure

  1. From the left navigation pane, select VDC > Virtual Desktop Pools > virtual desktop pool name.

  1. Click the Virtual Desktops tab.

  1. Click Reclaim Policy.

  1. In the dialog box that opens, select an action, specify the execution frequency and execution time, set whether to enable the reclamation policy after creation, and then click OK.

Parameters

  • Action: Select the action to take to reclaim floating desktops.

  • Shut Down: Shut down desktops.

  • Start: Start desktops.

  • Retrieve: Retrieve desktops.

  • Frequency: Select the execution frequency of the reclaim policy.

  • Yearly: Yearly execution.

  • Monthly: Monthly execution.

  • Weekly: Weekly execution.

  • Daily: Daily execution.

  • Start Date/End Date: Specify the start date and end date of each execution. Set this parameter for yearly, monthly, or weekly execution.

  • Execution Time: Specify the execution time or time range.

  • Enable Now: Set whether to enable the reclamation policy after creation.

Display user or user group information for a virtual desktop

Perform this task to display user or user group information for a virtual desktop.

Procedure

  1. From the left navigation pane, select VDC > Virtual Desktop Pools > virtual desktop pool name.

  1. Click the Virtual Desktops tab.

  1. Select a virtual desktop. The area at the bottom of the page displays the users or user groups that have access to the virtual desktop.

Parameters

  • Login Name/Group Name: Login name of a user, or the name of a user group. This parameter is empty if the virtual desktop is not assigned to any user or user group.

  • Username/Description: Username of a user, or the description of a user group. This parameter is empty if the virtual desktop is not assigned to any user or user group.

  • Type: Whether the item indicates a user or a user group.

  • Organization: Organization that the user or user group belongs to.

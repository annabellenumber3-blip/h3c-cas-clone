Manage VM templates of a virtual desktop pool

Perform this task to manage the VM templates of a virtual desktop pool. You can view network information and storage information for a VM template, deploy virtual desktops one by one, or bulk deploy virtual desktops.

Features

  • Deploy a virtual desktop

  • Bulk deploy fixed virtual desktops

  • Bulk deploy floating virtual desktops

  • Import virtual desktop configurations

  • Display VM template information for a virtual desktop pool

  • Display network and storage information for a VM template

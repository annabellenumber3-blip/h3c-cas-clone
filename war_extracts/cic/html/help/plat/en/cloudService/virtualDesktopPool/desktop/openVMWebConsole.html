Open the Web console of a virtual desktop

Perform this task to open the Web console of a virtual desktop and operate the virtual desktop. You can operate a running virtual desktop directly after you open its Web console. If you open the Web console of a shutdown virtual desktop, you must start the virtual desktop before you can operate it.

Restrictions and guidelines

  • If your browser blocks pop-up windows, you must disable the pop-up blocker manually.

  • When you use certain browsers, two consoles are opened for one virtual desktop. You can close one console.

  • To use the Web console, you must use IE 8.0, Chrome 14.0, or Firefox 10.0, or later.

Procedure

  1. From the left navigation pane, select VDC > Virtual Desktop Pools > virtual desktop pool name.

  1. Click the Virtual Desktops tab.

  1. Right-click the target VM and then select Console from the shortcut menu.

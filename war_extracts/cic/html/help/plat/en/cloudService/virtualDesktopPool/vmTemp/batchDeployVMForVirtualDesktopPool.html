Bulk deploy fixed virtual desktops

Perform this task to bulk deploy virtual desktops by using a VM template in a fixed virtual desktop pool.

Restrictions and guidelines

You can deploy a maximum of 200 virtual desktops at a time.

Procedure

  1. From the left navigation pane, select VDC > Virtual Desktop Pools > virtual desktop pool name.

  1. Click the VM Templates tab.

  1. Click the  icon in the Action column for a VM template.

  1. Select whether to enable the protection mode and quick deployment.

  1. Configure virtual desktops by using one of the following methods:

  • Manually Define: Manually configure virtual desktops one by one. For more information, see Deploy a virtual desktop.

  • Import from File: Import virtual desktop information from a file. For more information, see Import virtual desktop configurations.

  1. Click OK.

Parameters

  • Resource Pool Name: Resource pool where the virtual desktops belong.

  • Protection Mode: Set whether to enable the protection mode. In this mode, the incremental files of VM disks cannot be modified. If a user modifies the disk data of a VM, the data is automatically initialized when the VM restarts. If you enable the protection mode, quick deployment is also enabled.

  • Quick Deployment: Enable VM deployment based on the image files of the VM template. To shorten the VM creation time and save server storage space, select this option.

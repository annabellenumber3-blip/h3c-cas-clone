Add a virtual desktop pool

Perform this task to add a virtual desktop pool.

Restrictions and guidelines

Only CVM resource pools support virtual desktop pools. Before you create a virtual desktop pool, make sure the selected organization has CVM resource pools.

Procedure

  1. From the left navigation pane, select VDC > Virtual Desktop Pools.

  1. Click Add Virtual Desktop Pool.

  1. Enter the virtual desktop pool name, description, number of desktops, select the assignment mode, select whether to deploy VMs after desktop pool creation if you are creating a floating desktop pool, and then click Next.

  1. Select an organization, a VM template, a resource pool, and a storage. If you are creating a floating desktop pool, select a user group.

  1. If you select a Windows VM template, click Next. On the Peripheral tab, select a color depth, and specify whether to enable the local disk mapping, clipboard mapping, printer mapping, USB mapping, serial port mapping, and local audio mapping.

  1. Click Finish.

Parameters

  • Desktops: Enter the number of virtual desktops in the desktop pool.

  • For a fixed desktop pool, this parameter is the maximum number of assignable virtual desktops.

  • For a floating desktop pool, this parameter is the number of assigned virtual desktops.

  • Assignment Mode: Select the mode of virtual desktop assignment.

  • Fixed Desktop Pool: Virtual desktops in the desktop pool are manually assigned to users by an administrator. A user can use only the desktop assigned to him before the expiration time.

  • Floating Desktop Pool: Virtual desktops are automatically assigned to users by the system. When a user in the user group associated with a floating desktop pool logs in to SSV, the system automatically assigns a virtual desktop to the user. When the user logs out of SSV, the system automatically reclaims the virtual desktop.

  • Deploy Now: Set whether to deploy VMs immediately after the virtual desktop pool is created. Set this parameter when you are creating a floating desktop pool.

  • Organization: Select an organization.

  • Resource Pool: Select a resource pool.

  • VM Template: Select a VM template. Options are VM templates owned by the selected organization.

  • Storage: Select a storage pool. Options are storage pools that belong to the selected organization.

  • User Group: Select a user group. Options are user groups of the selected organization. Set this parameter when you are creating a floating desktop pool.

  • Color Depth: Select a color depth for remote sessions. Options include Best Quality (32 Bits), True Color (24 Bits), and High Color (16 Bits). Set this parameter when you use a Windows VM template.

  • Local Disk: Set whether to allow users to use local disks. Set this parameter when you use a Windows VM template.

  • Clipboard: Set whether to allow copy and paste operations between a client and a server. Set this parameter when you use a Windows VM template.

  • Printer: Set whether to allow users to use local printers. Set this parameter when you use a Windows VM template.

  • USB: Set whether to allow users to use local USB devices. Set this parameter when you use a Windows VM template.

  • Serial Port: Set whether to allow users to use local serial port devices. Set this parameter when you use a Windows VM template.

  • Local Audio: Set whether to allow users to use local audio devices. Set this parameter when you use a Windows VM template.

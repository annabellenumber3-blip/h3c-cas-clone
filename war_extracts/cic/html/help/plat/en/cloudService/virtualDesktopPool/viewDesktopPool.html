Display virtual desktop pool summary

Perform this task to display summary information for a virtual desktop pool, including configuration summary, desktop peripheral mapping rules, VM state statistics, top 5 virtual desktops by CPU usage, and top 5 virtual desktops by memory usage.

Procedure

From the left navigation pane, select VDC > Virtual Desktop Pools > virtual desktop pool name.

Parameters

  • Summary

  • Assignment Mode: Mode of virtual desktop assignment.

  • Max. VM Desktops: Maximum number of virtual desktops in the virtual desktop pool.

  • Resource Pool: Resource pool name.

  • Storage Pool: Storage pool name.

  • Organization: Organization name.

  • User Group: User group name. This parameter is displayed only for a floating desktop pool.

  • Desktop Peripheral Mapping Rule: This parameter is displayed only if a VM template installed with a Windows OS is used.

  • Color Depth: Color depth for remote sessions. Options include Best Quality (32 Bits), True Color (24 Bits), and High Color (16 Bits).

  • Local Disk: Whether to allow users to use local disks.

  • Clipboard: Whether to allow copy and paste operations between a client and a server.

  • Printer: Whether to allow users to use local printers.

  • USB: Whether to allow users to use local USB devices.

  • Serial Port: Whether to allow users to use local serial port devices.

  • Local Audio: Whether to allow users to use local audio devices.

  • VM State Statistics: Total number of VMs in the virtual desktop pool, and the number of VMs in each state, including Normal, Shutdown, and Abnormal.

  • Top 5 Virtual Desktops by CPU Usage: Top 5 virtual desktops with the highest CPU usage.

  • Top 5 Virtual Desktops by memory Usage: Top 5 virtual desktops with the highest memory usage.

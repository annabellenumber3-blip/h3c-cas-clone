Display virtual desktop information for a virtual desktop pool

Perform this task to display all virtual desktops in a virtual desktop pool.

Procedure

  1. From the left navigation pane, select VDC > Virtual Desktop Pools > virtual desktop pool name.

  1. Click the Virtual Desktops tab.

Parameters

  • State: State of the virtual desktop, including Running, Suspended, Shutdown, and Unknown.

  • CPU Usage: Average CPU usage over the past hour.

  • Memory Usage: Average memory usage over the past hour.

  • vCPUs: Number of CPU cores, which is the product of the number of CPUs and the number of cores per CPU.

  • Memory: Memory size.

  • OS: Operating system of the virtual desktop.

  • Protection Mode: Whether protection mode is enabled.

  • Anti-Virus: Whether anti-virus service is enabled for the virtual desktop.

  • Expire At: Expiration date of the virtual desktops.

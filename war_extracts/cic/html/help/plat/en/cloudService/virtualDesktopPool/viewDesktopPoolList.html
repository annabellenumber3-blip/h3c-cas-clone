Display virtual desktop pools

Perform this task to display all virtual desktop pools in CIC.

Procedure

From the left navigation pane, select VDC > Virtual Desktop Pools.

Parameters

  • Assignment Mode: Mode of virtual desktop assignment.

  • Fixed Desktop Pool: Virtual desktops in the desktop pool are manually assigned to users by an administrator. A user can use only the desktop assigned to him before the expiration time.

  • Floating Desktop Pool: Virtual desktops are automatically assigned to users by the system. When a user in the user group associated with a floating desktop pool logs in to SSV, the system automatically assigns a virtual desktop to the user. When the user logs out of SSV, the system automatically reclaims the virtual desktop.

  • VMs: Total number of VMs in the virtual desktop pool, and the number of VMs in each state, including Running and Shutdown.

  • Action: Actions that operators can perform on the virtual desktop pool.

  • To modify a virtual desktop pool, click the  icon.

  • To delete a virtual desktop pool, click the  icon.

  • To set the reclamation policy, click the  icon.

Implement a cloud disk workflow

Perform this task to implement a cloud disk workflow.

Procedure

  1. From the left navigation pane, select Workflows > Cloud Disk Tasks.

  1. Click the  icon in the Action column for the target cloud disk workflow.

  1. In the dialog box that opens, enter the size of the cloud disk and comment, and then click OK.

Parameters

  • Cloud Disk Name: Enter the name of the cloud disk that the terminal user has requested.

  • Size: Select the size of the cloud disk that the terminal user has requested.

  • Comment: Enter the comment on the workflow.

Display cloud disk workflow processing records

Perform this task to display cloud host workflow processing records.

Procedure

  1. From the left navigation pane, select Workflows > Cloud Disk Tasks.

  1. Select To Be Processed from the State list, and then click  in the Action column for the target cloud disk workflow.

Parameters

Workflow Info

  • Type: Type of the workflow.

  • Username: Name of the terminal user that has submitted the request.

  • Cloud Disk Name: Name of the cloud disk that the terminal user has defined when requesting the cloud disk.

  • Size: Size of the cloud disk that the terminal user has requested.

Approval Info

  • Operator: Operator that has processed the workflow.

  • Result: Processing result of the workflow.

  • Comment: Comment on the workflow.

  • Processed At: Time when the operator processed the workflow.

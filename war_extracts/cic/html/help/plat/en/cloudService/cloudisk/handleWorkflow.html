Process a cloud disk workflow

Perform this task to process cloud disk workflows in pending or being approved state.

Procedure

  1. From the left navigation pane, select Workflows > Cloud Disk Tasks.

  1. Select To Be Processed from the State list, and then click  in the Action column for the target cloud host workflow.

  1. Select a processing result and enter the comment. If you select Transfer, specify an operator.

  1. Click OK.

Parameters

Workflow Info

  • Type: Type of the workflow.

  • Username: Name of the terminal user that has submitted the request.

  • Cloud Disk Name: Name of the cloud disk that the terminal user has defined when requesting the cloud disk.

  • Size: Size of the cloud disk that the terminal user has requested.

Approval Info

  • Operator: Operator that has processed the workflow.

  • Result: Processing result of the workflow.

  • Comment: Comment on the workflow.

  • Processed At: Time when the operator processed the workflow.

Processing Info

  • Result: Select the processing result of the workflow. Options include Approve, Refuse, and Transfer.

  • Operator: Select an operator if you select Transfer.

  • Comment: Enter the comment on the workflow.

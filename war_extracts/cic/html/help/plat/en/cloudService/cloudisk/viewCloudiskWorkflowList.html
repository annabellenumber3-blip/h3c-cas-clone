Display cloud disk workflows

Perform this task to display cloud disk workflow summary.

Procedure

From the left navigation pane, select Workflows > Cloud Disk Tasks.

Parameters

  • Cloud Disk Name: Name of the cloud disk that the user has requested.

  • Username: Name of the terminal user.

  • State: State of the cloud disk workflow.

  • Size: Size of the cloud disk that the user has requested.

  • Action: Actions that operators can perform on the cloud disk workflow.

  • To process the cloud disk workflow, click the  icon.

  • To display the cloud disk workflow, click the  icon.

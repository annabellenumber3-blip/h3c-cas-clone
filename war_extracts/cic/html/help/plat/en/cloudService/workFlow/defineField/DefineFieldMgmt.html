Cloud host workflow management

Cloud host workflow management allows operators to view and process the workflows submitted by terminal users for requesting or canceling cloud hosts or extending the time to use the requested cloud hosts.

For cloud hosts installed with Unix operating systems, you can create only request workflows and the workflows can only be processed.

Features

  • Display cloud host workflows

  • Perform basic filtering for cloud host workflows

  • Perform advanced filtering for cloud host workflows

  • Process a cloud host workflow

  • Deploy VMs requested in a cloud host workflow

  • Download a cloud host workflow

  • Display cloud host workflow approval records

  • Customize cloud host workflow display information

  • Cloud host workflow additional field management

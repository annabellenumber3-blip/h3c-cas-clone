Add an additional field

Perform this task to add an additional field.

Restrictions and guidelines

  • You cannot add a new additional field when the number of additional fields reaches 40.

  • A new additional field has the lowest display priority.

Procedure

  1. From the left navigation pane, select Workflows > Cloud Host Tasks.

  1. Click Additional Field.

  1. In the dialog box that opens, click Add Field.

  1. Specify a field name and default value, and select a field property and type.

  • If you select Integer as the field type, enter the maximum and minimum values.

  • If you select Letters and Integers or Any Characters as the field type, enter the maximum length.

  • If you select List Options as the field type, enter an option in the Option field, and then click the  icon to add the option to the Available Options area.

  1. Click OK.

Parameters

  • Field Property: Specify whether the text field for the additional field can be empty or not. Options include Can Be Empty and Cannot be Empty.

  • Field Type: Specify the type of the value to be input in the text field. Options include Integer, Letters and Integers, Any Characters, and List Options.

  • Default Value: Enter the default value of the text field for the additional field.

  • If you select Integer as the field type, the default value must be an integer, and cannot be greater than the maximum value and smaller than the minimum value.

  • If you select Letters and Integers as the field type, the default value must be a combination of letters and integers, and the length cannot be greater than the maximum length.

  • If you select Any Characters as the field type, the default value can be any value, and the length cannot be greater than the maximum length.

  • If you select List Options as the field type, the default value must be a value in the Available Options field.

  • Max. Value: Enter the maximum value of the integer to be input in the text field. Only integers are allowed. The length cannot exceed nine integers and cannot be smaller than the minimum value. This parameter is required if you select Integer as the field type.

  • Min. Value: Enter the minimum value of the integer to be input in the text field. Only integers are allowed. The length cannot exceed nine integers and cannot be greater than the maximum value. This parameter is required if you select Integer as the field type.

  • Max. Length: Enter the maximum length of the integer to be input in the text field. Only integers are allowed. This parameter is required if you select Letters and Integers or Any Characters as the field type.

  • Option: Add an option to the list. This parameter is required if you select List Options as the field type.

  • Available Options: Values of the Option list. This parameter is required if you select List Options as the field type.

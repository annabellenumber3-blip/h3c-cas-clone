Customize cloud host workflow display information

Perform this task to customize information to be displayed on the cloud host workflow list.

Restrictions and guidelines

Make sure a minimum of one column is selected before saving the customized cloud host workflow configuration.

Procedure

  1. From the left navigation pane, select Workflows > Cloud Host Tasks.

  1. Click Customize.

  1. In the dialog box that opens, set the columns to display as required.

  • To add a parameter to the cloud host workflow, click the toggle button in the Display column for the target column name.

  • To remove a parameter from the cloud host workflow, click the toggle button in the Display column for the target column name.

  • To sort columns in the table, drag the target column names to the target locations as appropriate.

  1. Click the Column Width field for the target column name, and enter a column width.

  1. Click OK.

Parameters

Column Width: Enter a width for the column, in pixel.

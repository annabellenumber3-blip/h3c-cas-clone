Display additional fields

Perform this task to display the additional fields that an organization administrator requires a user to provide when the user applies for a cloud host.

Procedure

  1. From the left navigation pane, select Workflows > Cloud Host Tasks.

  1. Click Additional Field.

Parameters

  • Field Name: Name of the field.

  • Field Type: Type of the value to be input in the text field.

  • Field Property: Whether the additional field can be empty or not.

  • Action: Actions that operators can perform on the additional field.

  • To modify the additional field, click the  icon.

  • To delete the additional field, click the  icon.

Workflow management

Workflow management allows you to view and process workflows.

It provides the following features:

  • Cloud host workflow management: Processes cloud host application workflows submitted by terminal users, and assigns cloud resources to the users.

  • Cloud disk workflow management: Processes cloud disk application workflows submitted by terminal users, and assigns cloud resources to the users.

  • User pre-registration workflow management: Processes pre-registration application workflows submitted by terminal users, and assigns the users to organizations and user groups.

  • Cloud backup policy workflow management: Processes backup policy application workflows submitted by terminal users, and assigns backup policies to the users.

  • Workflow customization: Provides workflow customization for operators to improve flexibility of system resource application processing.

  • Cloud message management: Sends messages to terminal users.

  • Support ticket management: Replies to support tickets submitted by terminal users.

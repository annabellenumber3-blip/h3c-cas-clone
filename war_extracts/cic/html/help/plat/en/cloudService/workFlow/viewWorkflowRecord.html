Display cloud host workflow approval records

Perform this task to display cloud host workflow approval records.

Procedure

  1. From the left navigation pane, select Workflows > Cloud Host Tasks.

  1. Click the  icon in the Action column for the target cloud host workflow.

Parameters

Workflow Info

  • Alias: Alias of the cloud host requested or canceled by the user.

  • Cloud Disk: Cloud disk size the user has requested for the cloud host.

  • CPU: Number of CPUs the user has requested for the cloud host.

  • Memory Size: Memory size the user has requested for the cloud host.

  • Expire At: Time when cloud host requested by the user will expire.

  • Purpose: Reason why the user has requested the cloud host.

  • IP Allocation: IP allocation method for the cloud host that the user has requested.

Approval Info

  • Operator: Operator that has processed the workflow.

  • Result: Processing result of the workflow.

  • Comment: Comment on the workflow.

  • Processed At: Time when the operator processed the workflow.

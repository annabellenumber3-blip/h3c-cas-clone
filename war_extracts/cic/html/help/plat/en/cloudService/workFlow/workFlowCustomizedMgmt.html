Customize the workflow processing procedure

Perform this task to customize the workflow processing procedure. Workflows that can be customized include cloud host, cloud disk, user pre-registration, and cloud backup policy workflows.

Procedure

  1. From the left navigation pane, select Workflows > Customize Workflows.

  1. Click the  icon in the Action column for the target workflow type.

  1. Click the  icon to add an approval step and select required approvers.

  1. Click the  icon to select an operator.

  1. Click Save.

  1. To add more processing steps, repeat the previous step.

Parameters

  • Required Approvers: Select approvers required for the workflow to be approved. Options include Sole, All, Any, and Half.

  • Sole: The workflow is approved as long as one operator approves the workflow.

  • All: The workflow is approved only when all operators approve the workflow.

  • Any: The workflow is approved or refused as long as one of the operators approves or refuses the workflow.

  • Half: The workflow is approved only when half of the operators approve the workflow.

  • Operator: Select an approver at the current processing step.
